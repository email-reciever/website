## 技术资料

## 1\. [Stack Overflow 播客，采访 Node 作者 Ryan Dahl](https://link.zhihu.com/?target=https%3A//stackoverflow.blog/2024/03/19/why-the-creator-of-node-js-r-created-a-new-javascript-runtime/)

Dahl 分享了他的软件开发旅程，从开发 Deno 的初心，到 Deno 和 JavaScript 生态系统的未来。还介绍了 JSR，强调在 JavaScript 生态中安全的重要性。

## 2\. [React 编译器的类型系统](https://link.zhihu.com/?target=https%3A//www.recompiled.dev/blog/type-system/)

不了解 React 编译器的同学可以先阅读 [这篇文章](https://link.zhihu.com/?target=https%3A//react.dev/blog/2024/02/15/react-labs-what-we-have-been-working-on-february-2024%23react-compiler)，回过头来再深入阅读这篇博客。

官方的编译器长时间没有发布，社区已经等不及了，[React Unforget](https://link.zhihu.com/?target=https%3A//react-unforget.vercel.app/) 实现了类似的功能。

## 3\. [在现代应用中使用 localStorage 的全面指南](https://link.zhihu.com/?target=https%3A//rxdb.info/articles/localstorage.html)

探讨 localStorage API 的各个方面，包括优点、限制和现代应用程序中可用的替代存储选项。

## 4\. [给 JavaScript 开发者的 Docker 入门指南](https://link.zhihu.com/?target=https%3A//fly.io/javascript-journal/demystify-docker-js/)

搭配这篇一起看更过瘾：[让你一小时快速上手 Docker](https://link.zhihu.com/?target=https%3A//docker.easydoc.net/doc/81170005/cCewZWoN/lTKfePfP)

## 5\. [NG-Conf 2024](https://link.zhihu.com/?target=https%3A//ng-conf.org/)

Angular 在 AI 方面的探索包括文档聊天机器人、通过 Angular CLI 智能生成组件，同时 Angular 也正在和 Wiz 进行整合。

## 6\. [关于无障碍的常见误解](https://link.zhihu.com/?target=https%3A//a11ymyths.com/)

这个站点记录了大量关于无障碍的误解，无障碍影响的范围其实很大，影响着全球约 15% 的人口，即 10 亿人。
