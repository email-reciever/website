大家好，我是童欧巴，欢迎来到第 125 期技术周刊。

## 资讯

## [React Native 0.75](https://link.zhihu.com/?target=https%3A//reactnative.dev/blog/2024/08/12/release-0.75)

React Native 0.75 发布，带来了对百分比值的布局支持、新架构的稳定性改进、框架使用建议以及其他多项更新和性能优化。

## [Nuxt Scripts](https://link.zhihu.com/?target=https%3A//nuxt.com/blog/nuxt-scripts)

Nuxt Scripts 是 Nuxt 团队与 Google Chrome Aurora 团队合作开发的一款用于改善第三方脚本性能、隐私、安全和开发者体验的工具。

## [State of CSS 2024](https://link.zhihu.com/?target=https%3A//survey.devographics.com/en-US/survey/state-of-css/2024)

调查报告开启。

## [Waku 支持 React 19 server actions API](https://link.zhihu.com/?target=https%3A//waku.gg/blog/server-actions-are-here)

Waku v0.21 更新带来了对 React 19 服务器动作 API 的全面支持，使得开发者可以直接在 React 组件中执行服务器端逻辑，而无需手动创建 API 端点。

## [Node.js 自动化安全发布流程](https://link.zhihu.com/?target=https%3A//socket.dev/blog/node-js-doubles-security-releases-with-newly-automated-process)

Node.js 已经自动化了其安全发布流程，发布次数翻了一番，并正在与 Next 10 小组重新评估不受支持的实验性功能，以增强安全性。

## [微软发布 Phi-3.5 系列](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/X1kDlk0PBPvPXslcTJS4KA)

微软推出 Phi 3.5 系列 AI 模型，包括 Phi-3.5-mini-instruct、Phi-3.5-MoE-instruct 和 Phi-3.5-vision-instruct，分别擅长文本推理、多任务处理和视觉多模态任务，在多项基准测试中表现出色，甚至在某些情况下超越了 GPT-4o 和 Llama 3.1 等其他大型模型。

## [Hugging Face 发布适用浏览器的 SOTA 小模型](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/MXyrP0L7oVAjqJnrPqdnQg)

抱抱脸推出了在浏览器中运行的 SOTA 小模型，强调了严格过滤网络数据的重要性，并质疑了合成数据的普遍应用前景。

## [阿里开源通用多模态大模型 mPLUG-Owl3](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/QmxsPMe0wDElJwaUvwd9Tw)

阿里巴巴推出了通用多模态大模型 mPLUG-Owl3，该模型能在 4 秒内理解并回答有关长视频的问题，大幅提升了推理效率，同时在多个多模态场景的 Benchmark 中达到了最佳性能。

## [星尘智能发布人形机器人 Astribot S1](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/k81NE3bOxEIVw_3-eJY-Kg)

国产人形机器人 Astribot S1 展现了多项高级技能，如弹琴、泡茶、做华夫饼、演奏扬琴、投篮和远程逗猫等，其技术突破在于具身智能的数据获取、算法和算力的综合运用，以及刚柔耦合传动机构设计，使得机器人能够在安全性和操作精度上达到人类水平。

## [豆包大模型系列产品升级](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/QhW0WtCjT50eKAfqvVFiPg)

火山引擎发布了豆包大模型的升级版本，提升了语言理解和角色扮演能力，并整合了 RTC 技术，推出了对话式 AI 实时交互解决方案，使得用户与 AI 的语音交流更自然、流畅，并已在多个行业中应用。

## [OpenAI 上线 GPT-4o 微调](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/1iQmUOTHw_vOpFWkQ9zpUw)

OpenAI 推出 GPT-4o 微调功能，限时免费提供每日 100 万个训练 token，有助于开发者低成本构建应用程序，并提供了数据隐私和安全保障。

## 好文

## [React 正在成为一个全栈框架](https://link.zhihu.com/?target=https%3A//www.robinwieruch.de/react-full-stack-framework/)

React 正在从单纯的前端框架发展为全栈框架，通过引入 Server Components 和 Server Actions，它现在能够在前端和后端之间建立桥梁，为开发者提供了从 UI 到数据库的垂直特性实现能力。

## [正则表达式正在变得更好](https://link.zhihu.com/?target=https%3A//www.smashingmagazine.com/2024/08/history-future-regular-expressions-javascript/)

JavaScript 中的正则表达式在近年来得到了显著改进，现在已经具备了强大的功能，包括命名捕获组、环视、Unicode 属性等，并且未来还有更多提案正在逐步实现，以进一步增强其能力和易用性。

## [从 React 到 Effect](https://link.zhihu.com/?target=https%3A//effect.website/blog/from-react-to-effect)

探讨一下 Effect 的思维模型是如何映射到 React 中你熟知的概念的。
