![](https://pic4.zhimg.com/v2-01a0f3950943af7d9a423b5aed70ee87_b.jpg)

**2\.** [pnpm 9.5](https://link.zhihu.com/?target=https%3A//socket.dev/blog/pnpm-9-5-introduces-catalogs-shareable-dependency-version-specifiers)：pnpm 9.5 引入了“Catalogs”功能，实现了可共享的依赖关系版本说明，减少了合并冲突，改进了对 monorepos 的支持。使用时首先在 pnpm-workspace.yaml 定义 catalog，然后再 package.json 中使用，最终 package.json 会使用 workspace.yaml 中 catalog 定义的版本。

```text
# pnpm-workspace.yaml
catalogs:
  # Can be referenced through "catalog:react17"
  react17:
    react: ^17.0.2
    react-dom: ^17.0.2

  # Can be referenced through "catalog:react18"
  react18:
    react: ^18.2.0
    react-dom: ^18.2.0

{
  "name": "@example/components",
  "dependencies": {
    "react": "catalog:react18",
  }
}
```

**3\.** [Vitest 2.0](https://link.zhihu.com/?target=https%3A//github.com/vitest-dev/vitest/releases/tag/v2.0.0)：Vitest 2.0 发布，提供了更好的浏览器模式支持、新的 Blob Reporter 和一些稳定性改进。

## **开发工具**

**1\.** [es-toolkit](https://link.zhihu.com/?target=https%3A//es-toolkit.slash.page/zh_hans/)：es-toolkit 是一个新的 JavaScript 工具库，自称是更快、更现代化的 lodash 版本，而 lodash 则是更快、更现代化的 underscore 版本。

```text
import { countBy } from 'es-toolkit/array';

const array = [1, 2, 3, 4, 5, 6];
const result = countBy(array, x => x % 2 === 0 ? 'even' : 'odd');

console.log(result);
// 输出: { 'odd': 3, 'even': 3 }
```

**2\.** [Croner 8.1](https://link.zhihu.com/?target=https%3A//github.com/hexagon/croner)：定时任务，在后端中常见，Croner 支持全平台定时任务，包括 Node、Deno、Bun 和浏览器。

```text
const job = Cron('*/5 * * * * *', () => {
 console.log('每五秒运行一次');
});
```

**3\.** [file-type 19.1](https://link.zhihu.com/?target=https%3A//github.com/sindresorhus/file-type)：从 Buffer、Uint8Array 或 ArrayBuffer 中检测文件类型，现在还能从 streams 流中读取。

```text
import { fileTypeFromFile } from 'file-type';

console.log(await fileTypeFromFile('Unicorn.png'));
//=> {ext: 'png', mime: 'image/png'}
```
