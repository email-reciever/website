大家好，我是童欧巴，欢迎来到第 124 期技术周刊。

## 资讯

## [ViteConf 演讲者名单公布](https://link.zhihu.com/?target=https%3A//viteconf.org/)

今年 ViteConf 定档于 10 月 3 日 ～ 10 月 4 日，演讲者名单公布，依旧众星云集。

## [Vercel AI SDK 3.3](https://link.zhihu.com/?target=https%3A//vercel.com/blog/vercel-ai-sdk-3-3)

添加监控、多模态文件附件、useObject 支持将结构化对象生成流式传输到客户端实时显示 JSON 等。

## [Supabase 发布 postgres.new](https://link.zhihu.com/?target=https%3A//postgres.new/)

一个免费的工具，通过 Wasm 将 AI 连接到在浏览器中运行的 Postgres 数据库。

## [Sentry 宣布采用 Fair Source 模式](https://link.zhihu.com/?target=https%3A//blog.sentry.io/sentry-is-now-fair-source/)

Sentry 在 2024 年 8 月 6 日宣布采用 Fair Source 模式，这是一种新的软件共享方式，旨在平衡公司控制产品路线图和商业模式的需求，以及开发者的自由和可持续性。

## [Puppeteer 宣布对 Firefox 官方支持](https://link.zhihu.com/?target=https%3A//hacks.mozilla.org/2024/08/puppeteer-support-for-firefox/)

Puppeteer 23 版本正式支持 Firefox，使得开发者可以使用 Puppeteer 进行跨浏览器的自动化和端到端测试。

## [管理您的 JavaScript 命令行工具](https://link.zhihu.com/?target=https%3A//github.com/volta-cli/volta)

Volta 用 Rust 编写，用于管理和部署 JavaScript 工具链。它的特点是快速、可靠、通用，支持跨平台，并且能够在不同的包管理器和 Node.js 运行时之间无缝工作。

## [xAI 发布 Grok 2](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/nBaY2srcMSzvEoecOyh1Cg)

xAI 推出了新一代的 AI 模型 Grok-2，该模型在生成图像、识别图像、聊天、推理和编码等方面表现出色，甚至能与 GPT-4o 相媲美，马斯克对其发展表示赞赏。

## [Cosine 发布自主 AI 工程师 Genie](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/9Gz9vc6IfKkpvLOp-J0NOg)

基于 GPT-4o，初创公司 Cosine 开发的 AI 工程师 Genie 超越了 Devin，成为了地表最强的「AI 软件工程师」，能够自主思考和行动，与人类工程师并肩作战。

## [佐治亚理工学院和 IBM 研究院发布 Transformer Explainer](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/vLyIrRyoWYjhMN4gTRgA6g)

佐治亚理工学院和 IBM 研究院研究者开发的一款基于 web 的开源交互式可视化工具「Transformer Explainer」，该工具旨在帮助非专业人士了解 Transformer 模型的工作原理，并支持本地运行 GPT-2 进行实时推理。

## [腾讯优图实验室开源首个多模态大语言模型 VITA](https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/_ILH_GZaKMUlRaqsZBU0Eg)

VITA 是第一个开源的多模态大语言模型，能够处理视频、图像、文本和音频模态，并具备先进的多模态交互体验，其性能在多个基准测试中得到了验证。

## 好文

## [谷歌 Angular 负责人谈 JavaScript 框架的融合](https://link.zhihu.com/?target=https%3A//thenewstack.io/google-angular-lead-sees-convergence-in-javascript-frameworks/)

Google 的 Angular 主管 Minko Gechev 认为，Angular 和 React 等前端框架正在逐渐融合，趋向于使用相同的技术和特性。

## [Svelte 5 中的细粒度反应性](https://link.zhihu.com/?target=https%3A//frontendmasters.com/blog/fine-grained-reactivity-in-svelte-5/)

主要介绍了 Svelte 5 中的细粒度反应性，并通过示例和代码演示了如何在 Svelte 中实现对象和数组的细粒度反应性，以及如何通过类和工具函数来控制数据的反应性。

## [JavaScript 中内存泄漏的常见原因](https://link.zhihu.com/?target=https%3A//www.trevorlasn.com/blog/common-causes-of-memory-leaks-in-javascript)

主要讲解了 JavaScript 中常见的内存泄漏原因及如何识别和解决它们，并提供了 Node.js 和 Deno.js 环境下的内存管理和性能分析方法。

## [是时候谈谈“CSS5”了吗？](https://link.zhihu.com/?target=https%3A//www.smashingmagazine.com/2024/08/time-to-talk-about-css5/)

主要讨论了 CSS 发展的现状与未来，特别是 CSS3 之后的版本命名问题，以及 CSS-Next 社区组如何致力于推动 CSS 特性的分类与命名，以帮助开发者、教育者和雇主更好地理解和推广 CSS 的进步。
