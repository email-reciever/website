**5\. [ECMAScript 安全赋值操作符提案](https://link.zhihu.com/?target=https%3A//github.com/arthurfiorette/proposal-safe-assignment-operator)**：该提案提出了一种有趣的语法（?=），可从赋值中返回一个 `[error, value]` 元组。  

```text

```

## **开发工具**

**1\. [Volta 2.0](https://link.zhihu.com/?target=https%3A//volta.sh/)**：这是一款由 Rust 支持的老牌工具，用于安装和切换 JavaScript 相关工具（如 Node、TypeScript、Yarn 等）。

```text
volta list node // 查看 volta 已安装的 node 版本
volta install node@<version> // 全局安装(安装并切换到当前版本)
volta pin node@<version> // 项目中安装(当在项目中，执行该命令之后，会在项目的 package.json 中)
```

**2\. [jscodeshift 17.0](https://link.zhihu.com/?target=https%3A//github.com/facebook/jscodeshift)**：来自 Facebook 的 JavaScript codemod 工具包。

```text
jscodeshift(src)
  .find(jscodeshift.Identifier)
  .forEach(function(path) {
    // do something with path
  });
```

**3\. [tailwindcss-highlights](https://zhuanlan.zhihu.com/github.com/jln13x/tailwindcss-highlights)**：为文本添加高光的 Tailwind 插件 - 这里的 “高光 ”指的是一种类似于荧光笔的效果，用于在纸质文档上做标记，包括 20 种不同的效果。
