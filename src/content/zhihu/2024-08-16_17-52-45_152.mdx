**栗子前端技术周刊第 39 期 (2024.7.15 - 2024.7.21)**：浏览前端一周最新消息，学习国内外优秀文章视频，让我们保持对前端的好奇心。

## **技术资讯**

**1\.** [Node.js v22.5.0](https://link.zhihu.com/?target=https%3A//nodejs.org/zh-cn/blog/release/v22.5.0)：Node.js v22.5.0 (Current) 发布，有两点值得注意，一是 `node:http` 中的 WebSocket 功能现在已公开，二是 Node 已内嵌 SQLite，现在可通过 `node:sqlite` 直接访问 SQLite。

```text
import { DatabaseSync } from 'node:sqlite';
const database = new DatabaseSync(':memory:');

// 执行 SQL 语句
database.exec(`
  CREATE TABLE data(
    key INTEGER PRIMARY KEY,
    value TEXT
  ) STRICT
`);
```

**2\.** [Deno 1.45](https://link.zhihu.com/?target=https%3A//deno.com/blog/v1.45)：Deno 1.45 已发布，除了继续大幅改进 Node.js 兼容性，此版本还支持了 workspaces 和 monorepos，即单仓多模块。

```text
{
  "workspace": ["./packages/A", "./packages/B"]
}
```

**3\.** [博客园求救信](https://link.zhihu.com/?target=https%3A//www.cnblogs.com/cmt/p/18302049)：2024 年第三季度的资金缺口将博客园推到命运的关口，博客园向开发者求救，求开博客园会员。博客园的初衷是为几百万程序员开发者创建一番净土，专心为开发者服务，尽量不接广告，这点比起某 DN 平台的各种骚操作来说，确实好非常多。  

**4\.** [jQuery 4.0 Beta 2](https://link.zhihu.com/?target=https%3A//blog.jquery.com/2024/07/17/second-beta-of-jquery-4-0-0/)：今年二月，jQuery 发布了 4.0.0 的第一个测试版。现在 jQuery 已经准备好发布第二个测试版，同时预计很快就会发布候选版本。Beta 2 版本对 jQuery 的测试框架进行了重大重写，删除了所有过时或不支持的依赖项。除此之外，还修复了导出问题，保证包能在 ESM 和 CommonJS 下使用，更多内容可见 Changelog。
