**栗子前端技术周刊第 37 期 (2024.7.1 - 2024.7.7)**：浏览前端一周最新消息，学习国内外优秀文章视频，让我们保持对前端的好奇心。

## **技术资讯**

**1\.** [Mako 开源](https://link.zhihu.com/?target=https%3A//makojs.dev/blog/mako-open-sourced_zh-CN): Mako 是「极快」和「生产级」的前端构建工具，基于 Rust。Mako 自 2023.11.24 起已在蚂蚁内部正式发布，通过工程化的方式验证了数千个项目和所有用到的 npm 包及其不同版本，已落地数百个项目，并对内服务了中后台、小程序、H5 移动端、低代码、营销、组件库、组件打包、Serverless Function 等多个不同平台和业务场景，已具备了生产级的能力。

![](https://pic4.zhimg.com/v2-19840e82c55f8214abc3e718bc9801ff_b.jpg)

**2\.** [Polyfill 攻击波及 10 万多个网站](https://link.zhihu.com/?target=https%3A//sansec.io/research/polyfill-supply-chain-attack)：polyfill.js 是一个支持旧版浏览器的流行开源库，超过 10 万个网站使用 `cdn.polyfill.io` 域名嵌入了该库，它通过读取请求的 User-Agent，并返回该特定浏览器所需的 polyfills。今年 2 月，一家中国公司购买了该 CDN 的 GitHub 仓库和域名，并开始在使用该服务的 10 万多个网站上注入 `www.googie-anaiytics.com` 的虚假分析脚本。所以值得借鉴的是，在做前端性能优化的时候，你可能会把一些资源通过 CDN 加载，但是要使用可靠 CDN 服务，最好是自己公司的。

**3\.** [使用 Expo 创建 RN 应用](https://link.zhihu.com/?target=https%3A//reactnative.dev/blog/2024/06/25/use-a-framework-to-build-react-native-apps)：截至今天，React Native 唯一推荐的社区框架是 Expo。Expo 从 React Native 早期就开始投资建设 React Native 生态系统，所以 React Native 团队相信 Expo 提供的开发体验是同类中最好的。

**4\.** [State of JavaScript 报告中去年前端技术亮点](https://link.zhihu.com/?target=https%3A//2023.stateofjs.com/zh-Hans/awards/)：2023 的 JS 报告调查中，使用率增加最大、满意率最高和正面评价比例最高的技术都是 **Vite**，讨论次数最多的库则为 **React**，其他更多信息可查看官网报告。
