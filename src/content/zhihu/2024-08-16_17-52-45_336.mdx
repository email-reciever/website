**栗子前端技术周刊第 42 期 (2024.8.5 - 2024.8.11)**：浏览前端一周最新消息，学习国内外优秀文章视频，让我们保持对前端的好奇心。

## **技术资讯**

**1\. [Puppeteer 支持 Firefox](https://link.zhihu.com/?target=https%3A//hacks.mozilla.org/2024/08/puppeteer-support-for-firefox)**：Puppeteer 现在已支持 Firefox。这意味着你用 Puppeteer 编写的自动化端到端测试用例，可以同时在 Chrome 浏览器和 Firefox 浏览器上运行。

```text
import puppeteer from "puppeteer";

const browser = await puppeteer.launch({
  browser: "firefox"
});

const page = await browser.newPage();
// ...
await browser.close();
```

**2\. [Astro 4.13](https://link.zhihu.com/?target=https%3A//astro.build/blog/astro-4130/)**：Astro 4.13 将请求重写和内容收集 JSON 模式从实验性功能正式加入到稳定功能中，除此之外，还有一些日志改进。

**3\. [nvm 0.40](https://link.zhihu.com/?target=https%3A//nodeweekly.com/link/158438/fcd03620c9)**：Node 版本管理命令行工具，0.40 版本 `nvm use/nvm install` 增加了 `--save` 选项，可将操作写到 `.nvmrc` 文件中，除此之外，还支持了 `NVM_AUTH_HEADER` 环境变量和 `.nvmrc` 文件支持写注释。

## **技术文章**

**1\. [The Node.js best practices list](https://link.zhihu.com/?target=https%3A//github.com/goldbergyoni/nodebestpractices%23readme)**：2024 年版面向 Node 开发人员的深度指南。该指南分为 8 个部分并定期更新，深入探讨了从错误处理和代码风格到 Docker 和安全实践等多个领域。  

**2\. [Reading from the Clipboard in JavaScript](https://link.zhihu.com/?target=https%3A//blog.logrocket.com/javascript-design-patterns-guide/)**：浏览器对用户剪贴板的读写有很好的支持，这为在 web 上获得更好的体验提供了可能，本文将介绍如何使用剪贴板。  

**3\. [告别轮询，SSE 流式传输可太香了](https://link.zhihu.com/?target=https%3A//juejin.cn/post/7355666189475954725)**：本文将从 SSE 的概念、与 Websocket 对比、SSE 应用场景多个方面介绍 SSE 流式传输。对于需要轮询的业务场景来说，采用 SSE 在对应场景下会是一个更好的技术方案。
