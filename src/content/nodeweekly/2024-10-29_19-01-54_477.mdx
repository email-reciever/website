[Transformers.js v3: Now You Can Run Transformers in Node.js](https://nodeweekly.com/link/161508/0a4070b383 "huggingface.co") — [A JavaScript port](https://nodeweekly.com/link/161509/0a4070b383) of Hugging Face’s `transformers` Python library that makes it possible to run natural language, vision, and audio machine learning models. v3 adds WebGPU support and now supports Node (_plus Deno and Bun_) as well as the browser. [1200+ models](https://nodeweekly.com/link/161510/0a4070b383) are ready to run in areas like embeddings, text generation, and speech recognition (as with [whisper-small](https://nodeweekly.com/link/161511/0a4070b383)).

[Node v22.11.0 'Jod' Released; Becomes Active LTS Release](https://nodeweekly.com/link/161512/0a4070b383 "nodejs.org") — With this release, Node 22 flips from being the ‘Current’ cutting edge release to being a dependable release with the all-important LTS (Long Term Support) designation. It is, however, broadly the same as Node v22.10.

[Node v23.1.0 (Current) Released](https://nodeweekly.com/link/161513/0a4070b383 "nodejs.org") — Node 22 takes a breather, but Node 23 picks up the ‘Current’ baton and will get all the juiciest new features until Node 24 arrives in April/May 2025. In v23.1, JSON modules and import attributes are now stable, the `MockTimers` test runner API is stable, and `Buffer` objects created from resizable `ArrayBuffer`s will now update their size properly.
