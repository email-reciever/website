[Node.js Now Supports TypeScript By Default](https://nodeweekly.com/link/163868/0a4070b383 "www.totaltypescript.com") — It’s been possible to run TypeScript directly with Node.js since v22.6 introduced [experimental ‘type stripping’ support](https://nodeweekly.com/link/163869/0a4070b383), but in the [to-be-released-at-any-moment Node 23.6](https://nodeweekly.com/link/163870/0a4070b383) (or the current Node nightly) you’ll be able to run `node yourapp.ts` and it’ll Just Work™. Matt provides some details on _how_ it works and what you’ll be able to do.
