[Node 24 (Current) Released](https://nodeweekly.com/link/168866/0a4070b383 "nodejs.org") — [Node’s release lines](https://nodeweekly.com/link/168867/0a4070b383) are in flux – v18 has just gone EOL and now v23 bows out to let v24 be the ‘Current’ release for when you want all the cutting edge features. It comes with npm 11, V8 13.6 (hello `RegExp.escape`, `Float16Array`, `Error.isError` and more), the [`URLPattern` API](https://nodeweekly.com/link/168868/0a4070b383) exposed as a default object, and Undici 7.

Node.js Team

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/23b42ef7.png)](https://nodeweekly.com/link/168826/0a4070b383)

[Create Your Own AI Agent](https://nodeweekly.com/link/168826/0a4070b383 "frontendmasters.com") — Join Scott Moss for this detailed video course and discover how to build an AI agent from scratch. You'll explore LLMs, chat history management, tokens, memory, agent loops, and much more — coming away with know-how and resources for building modern agent-based API apps.

Frontend Masters sponsor

[Take the Latest Node.js _Next 10_ Survey](https://nodeweekly.com/link/168827/0a4070b383 "linuxfoundation.research.net") — The Node.js core team and the Linux Foundation are keen for you to take part in the latest Node.js Next 10 Survey. [‘Next 10’](https://nodeweekly.com/link/168828/0a4070b383) is a term the Node team uses to refer to strategizing Node’s direction over the next ten years, and the survey helps them get your input into this.

The Linux Foundation

**IN BRIEF:**

*   [Bun v1.2.12 is out](https://nodeweekly.com/link/168829/0a4070b383) with, surprise, surprise, yet more Node.js compatibility enhancements :-) As part of Bun's new full-stack focus, you can now also stream _browser_ console logs back through Bun at the terminal.
    
*   In other alternative runtime news, [Deno 2.3 has been released](https://nodeweekly.com/link/168830/0a4070b383) with improvements to its single-binary compilation feature which now supports FFI and Node native add-ons, as well as support for using local npm packages.
    
*   The OpenJS Foundation has [announced its new board of directors](https://nodeweekly.com/link/168831/0a4070b383) which includes numerous JavaScript and Node luminaries including Jordan Harband and Matteo Collina.
    

[Retrieval-Augmented Generation (RAG) with Llama Stack and Node](https://nodeweekly.com/link/168832/0a4070b383 "developers.redhat.com") — Meta’s [Llama Stack](https://nodeweekly.com/link/168833/0a4070b383) is a unified set of APIs for working with numerous parts of the modern LLM-powered stack, including RAG, as demonstrated here.

Michael Dawson

💡 Michael presents a broader overview of Llama Stack in: [A practical guide to Llama Stack for Node.js developers.](https://nodeweekly.com/link/168834/0a4070b383)

[Converting Values to Strings in JavaScript](https://nodeweekly.com/link/168835/0a4070b383 "2ality.com") — When Dr. Axel says _“converting values to strings in JavaScript is more complicated than it might seem”_, I’m going to believe him. An interesting poke around into something simple that you might not think about very much.

Dr. Axel Rauschmayer

[Plug Enterprise SSO to Your Node App in Minutes](https://nodeweekly.com/link/168836/0a4070b383 "www.scalekit.com") — Add SAML and OIDC SSO on top of your existing auth using Scalekit’s Node SDK — no rewrites required. [Learn more](https://nodeweekly.com/link/168836/0a4070b383).

*   [SVGO 4.0.0 RC4](https://nodeweekly.com/link/168853/0a4070b383) – Node.js-powered SVG optimizer.
    
*   [Jira.js 5.1](https://nodeweekly.com/link/168854/0a4070b383) – A wrapper for Jira's numerous APIs.
    
*   [noblox.js 6.2](https://nodeweekly.com/link/168855/0a4070b383) – A Node.js API wrapper for Roblox.

📢  Elsewhere in JavaScript

A roundup of some other interesting stories in the broader JavaScript landscape, in case you've missed them:

*   Currently at stage 3 at TC39, the [ECMAScript Explicit Resource Management proposal](https://nodeweekly.com/link/168859/0a4070b383) has been implemented in Chrome 134+ and Node 24.
    
*   Google Apps Script brings a lot of JavaScript power to automating actions across Google's various services, but I find it a little opaque sometimes, so [this tutorial on exporting Google Analytics data to Google Sheets](https://nodeweekly.com/link/168860/0a4070b383) is refreshing.
    
*   The popular commercial GSAP animation library [has become free to use for all](https://nodeweekly.com/link/168861/0a4070b383), including commercial use cases. It always amazes me [what it can do.](https://nodeweekly.com/link/168862/0a4070b383)
    
*   Josh Comeau's [Operator Lookup](https://nodeweekly.com/link/168863/0a4070b383) site makes it easy to look up with a JavaScript operator does, or even find ones you haven't encountered before, like [\>>>=](https://nodeweekly.com/link/168864/0a4070b383) perhaps.
    
*   If you've been avoiding Redis after the license controversy in 2024, you'll be happy to know [Redis is now open source again](https://nodeweekly.com/link/168865/0a4070b383) as of version 8.0.
