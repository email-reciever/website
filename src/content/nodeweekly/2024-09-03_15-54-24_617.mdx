[yocto-spinner: Tiny Terminal Spinner](https://nodeweekly.com/link/159203/0a4070b383 "github.com") — Fresh from the one-man module powerhouse that is Sindre Sorhus comes a new project: a tiny, as simple-as-possible terminal spinner/progress control. _(‘Yocto’ is a [metric prefix](https://nodeweekly.com/link/159204/0a4070b383) that sits below nano, pico, femto, atto, and even zepto..)_

[google-spreadsheet: Google Sheets API Wrapper for Node](https://nodeweekly.com/link/159205/0a4070b383 "theoephraim.github.io") — Work with Google’s online spreadsheet from Node, whether at the cell, row, worksheet or document level. Theo Ephraim

[Light My Request: Fake HTTP Injection Library](https://nodeweekly.com/link/159207/0a4070b383 "github.com") — Injects a fake HTTP request/response into a Node HTTP server for simulating server logic, writing tests, or debugging. Doesn’t use a socket connection so can be run against an inactive server (server not in listen mode).

Fastify

Sindre Sorhus

*   [Discord.js 14.16](https://nodeweekly.com/link/159214/0a4070b383) – Official library for interacting with Discord's API.
    
*   [NVM Desktop 3.4](https://nodeweekly.com/link/159215/0a4070b383) – Desktop UI for [Node Version Manager](https://nodeweekly.com/link/159216/0a4070b383).
