Master.dev sponsor

[Node.js 26.3.1, 24.17.0, and 22.23.0 Released](https://nodeweekly.com/link/186725/0a4070b383 "nodejs.org") — The latest [round of security releases](https://nodeweekly.com/link/186726/0a4070b383) is here, with [26.3.1 (Current)](https://nodeweekly.com/link/186725/0a4070b383), [24.17.0 (LTS)](https://nodeweekly.com/link/186727/0a4070b383), and [22.23.0 (LTS)](https://nodeweekly.com/link/186728/0a4070b383) fixing _eleven_ vulnerabilities. Two are of 'high' severity: one around [normalizing hostnames for server identity checks](https://nodeweekly.com/link/186729/0a4070b383) in `tls`, and the other in [WebCrypto](https://nodeweekly.com/link/186730/0a4070b383).

Antoine du Hamel

🔒 The releases above also include [Undici 8.5.0](https://nodeweekly.com/link/186763/0a4070b383) which has addressed some security advisories of its own.

**IN BRIEF:**

*   As [npm prepares to disable install scripts](https://nodeweekly.com/link/186731/0a4070b383) by default, Andrew Nesbitt looks at [how other tools and ecosystems handle install-script allow-listing](https://nodeweekly.com/link/186732/0a4070b383).
    
*   [Iroh 1.0](https://nodeweekly.com/link/186733/0a4070b383) is a peer-to-peer encrypted networking stack that punches through NAT and firewalls. It's built in Rust but has [official Node.js bindings](https://nodeweekly.com/link/186734/0a4070b383).
    
*   [Babel 8.0 has been released](https://nodeweekly.com/link/186735/0a4070b383): no new features, but with a focus on modernization. It's ESM-only, ships TypeScript types, and no longer compiles to ES5 and CommonJS by default.
    

[Shipping `psql` Without `psql`: A Pure-TypeScript Postgres Client](https://nodeweekly.com/link/186736/0a4070b383 "neon.com") — Postgres platform _Neon_ wanted the experience of Postgres’ `psql` client in their own Node-powered CLI, without the binary dependency. Here’s how they did it using AI, a strong conformance harness, and a security review.

Vadim Kharitonov (Neon)

[A Backdoor in a LinkedIn Job Offer](https://nodeweekly.com/link/186737/0a4070b383 "roman.pt") — A hacker, posing as a recruiter on LinkedIn, asked the author to diagnose an issue in a Node project, but the dependencies had a backdoor included. Both the recruiter and the git repo involved also used real identities of people unrelated to the scam.

Roman Imankulov

📄 [Wasp Now Lets You Write Your Full-Stack Logic as a Spec in TypeScript](https://nodeweekly.com/link/186741/0a4070b383) – Say goodbye to the React + Node.js + Prisma full-stack framework’s custom `.wasp` DSL. Martin Sosic (Wasp)

📄 [Why pnpm No Longer Expands Environment Variables in `.npmrc`](https://nodeweekly.com/link/186742/0a4070b383) Zoltan Kochan

🛠 Code & Tools

✉️ [Nodemailer 9.0: Easily Send Emails from Node.js](https://nodeweekly.com/link/186743/0a4070b383 "nodemailer.com") — A long-standing, popular way to send emails over SMTP, through SES, or Sendmail. v9.0 now validates TLS connections by default when fetching remote content.

Andris Reinman

[zod-compiler: Compile Zod Schemas Into Zero-Overhead Validators](https://nodeweekly.com/link/186744/0a4070b383 "github.com") — A build-time compiler that plugs into Vite, webpack, et al. and rewrites existing schemas into optimized, tree-shakeable validators, with the full Zod API preserved.

Gajus Kuizinas

💬 [Node Telegram Bot API 1.0/1.1](https://nodeweekly.com/link/186745/0a4070b383 "github.com") — Interact with the official [Telegram Bot API](https://nodeweekly.com/link/186746/0a4070b383) from Node. Now modernized, rewritten in TypeScript, and ESM-only, with Telegram Bot API 10.1 support.

Yago

[eslint-plugin-unicorn 67.0: 200+ Powerful Linting Rules](https://nodeweekly.com/link/186747/0a4070b383 "github.com") — Sindre’s set of rules that will seriously challenge (and improve) your code, whether it's [comments](https://nodeweekly.com/link/186748/0a4070b383), [the depth of nested calls](https://nodeweekly.com/link/186749/0a4070b383), [preferring `Temporal` over `Date`](https://nodeweekly.com/link/186750/0a4070b383), and more.

Sindre Sorhus et al.
