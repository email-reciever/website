Microsoft | AMD sponsor

[You Can't Cancel a Promise (Except Sometimes You Can)](https://nodeweekly.com/link/183455/0a4070b383 "www.inngest.com") — You can’t cancel a promise, but you can halt an async function by making it `await` a promise that never resolves. The function silently stops, and GC cleans up after it. It’s not just a trick; Inngest uses it in production to interrupt long-running workflow functions.

Aaron Harper (Inngest)

**IN BRIEF:**

*   Google's compilers team has [proposed a high-level IR for JavaScript](https://nodeweekly.com/link/183456/0a4070b383) and shared a tool ([JSIR](https://nodeweekly.com/link/183457/0a4070b383)) it's already using internally to take advantage of it for analysis and source-to-source transformation.
    
*   Microsoft has announced the [end of support for Node 20.x](https://nodeweekly.com/link/183458/0a4070b383) in the _Azure SDK for JavaScript._
    
*   [npm trusted publishing now supports CircleCI](https://nodeweekly.com/link/183459/0a4070b383) as an OIDC provider.
    
*   🎤 [A 50-minute chat](https://nodeweekly.com/link/183460/0a4070b383) (with transcript) from two of the developers behind the [npmx project](https://nodeweekly.com/link/183461/0a4070b383) — an increasingly popular way to browse the npm registry.
    

[Node's Security Bug Bounty Program Paused Due to Loss of Funding](https://nodeweekly.com/link/183462/0a4070b383 "nodejs.org") — Since 2016, the Node.js project has offered bounties for qualifying security vulnerability reports. This was funded by the _Internet Bug Bounty_ program which is [on hiatus](https://nodeweekly.com/link/183463/0a4070b383) as it figures out its role in an AI-assisted landscape. Reports can still be made, but with no monetary reward.
