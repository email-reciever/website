[Node Moves to Enable Temporal By Default](https://nodeweekly.com/link/183842/0a4070b383 "github.com") — The [Temporal API](https://nodeweekly.com/link/183843/0a4070b383), designed to modernize JavaScript’s date/time handling, [reached stage 4](https://nodeweekly.com/link/183844/0a4070b383) last month. Node was waiting on V8 to make it enabled by default, which happened in V8 14.4, and the wheels are now in motion for an eventual release in Node 26.

📢  Elsewhere in the ecosystem

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/ulgnvyrw2bt4ykhbefbx.jpg)](https://nodeweekly.com/link/183874/0a4070b383)

*   [FluidCAD](https://nodeweekly.com/link/183874/0a4070b383) _(above)_ is a new project bringing parametric CAD to the JavaScript world where you can write code to create/manipulate objects and see a live update of what you're making. It's built on top of [OpenCascade.js](https://nodeweekly.com/link/183875/0a4070b383), a JS/WASM port of the open source [OpenCascade](https://nodeweekly.com/link/183876/0a4070b383) 3D geometry library.

*   Ally Piechowski shares [the `git` commands she runs before reading any code](https://nodeweekly.com/link/183879/0a4070b383). They provide a handy way to get a quick grasp on what's happening in larger, multi-contributor repos.
    
*   A cautionary read for anyone running Node on AWS Lambda with MySQL or MariaDB: [The AWS Lambda 'Kiss of Death'](https://nodeweekly.com/link/183880/0a4070b383) walks through how connection reuse can leave transactions dangling, causing InnoDB's history list to grow unchecked until the database freezes.
    
*   GitHub has [opened a private preview of _Stacked PRs_](https://nodeweekly.com/link/183881/0a4070b383), giving native support for a workflow where one big change is split into a chain of dependent PRs.
