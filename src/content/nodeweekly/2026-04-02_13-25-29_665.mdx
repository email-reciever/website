[Node.js 25.9.0 (Current) Released](https://nodeweekly.com/link/183181/0a4070b383 "nodejs.org") — Including a `--max-heap-size` option to set the maximum heap size for a process, James Snell’s experimental [‘better streams API’](https://nodeweekly.com/link/183182/0a4070b383) implementation lands as `stream/iter` ([docs here](https://nodeweekly.com/link/183183/0a4070b383)), plus test runner module mocking improvements.

🛠 Code & Tools

🤖 [Transformers.js v4: Run AI Models from JavaScript](https://nodeweekly.com/link/183200/0a4070b383 "github.com") — Run [Hugging Face](https://nodeweekly.com/link/183201/0a4070b383)\-hosted models to do things like LLMs, vision, and audio models from Node, in the browser, etc. v4 switches to WebGPU and is installable with npm. There are [many demos](https://nodeweekly.com/link/183202/0a4070b383) covering [speech transcription](https://nodeweekly.com/link/183203/0a4070b383), [using Qwen 3.5](https://nodeweekly.com/link/183204/0a4070b383), and [video captioning.](https://nodeweekly.com/link/183205/0a4070b383)

Hugging Face

💡 The browser demos use large model downloads, so server-side use with Node may be the best use case, even if not the _coolest_.

[node-re2: Bindings for Google's RE2 Regex Library](https://nodeweekly.com/link/183206/0a4070b383 "github.com") — [RE2](https://nodeweekly.com/link/183207/0a4070b383) is a regular expression library with linear-time matching, making it immune to [ReDoS](https://nodeweekly.com/link/183237/0a4070b383) attacks caused by backtracking. node-re2 offers it as a near drop-in replacement for `RegExp`.

Eugene Lazutkin

[Defuddle: Extract the Main Content from Pages](https://nodeweekly.com/link/183209/0a4070b383 "github.com") — Strips clutter from HTML leaving only the primary content for you to use. There’s [a demo](https://nodeweekly.com/link/183210/0a4070b383) where you can try it out.

Steph Ango

*   [dotenv 17.4](https://nodeweekly.com/link/183213/0a4070b383) – Loads environment variables from `.env` files into `process.env`. While now [a built-in Node feature](https://nodeweekly.com/link/183214/0a4070b383), dotenv is a popular dependency with 100M+ downloads a week and offers finer-grained control over the process.
    
*   [numpy-ts 1.2](https://nodeweekly.com/link/183215/0a4070b383) – NumPy implementation for TS/JS. Now at ~50% native perf and with Float16 support. ([Homepage](https://nodeweekly.com/link/183216/0a4070b383))
    
*   [Path-to-RegExp 8.4](https://nodeweekly.com/link/183217/0a4070b383) – Library to convert path strings with parameters into regexes.
    
*   [globby 16.2](https://nodeweekly.com/link/183218/0a4070b383) – Promise-based glob matching, now with [global gitignore file support.](https://nodeweekly.com/link/183219/0a4070b383)
    
*   [ts-blank-space 0.8](https://nodeweekly.com/link/183220/0a4070b383) – Pure JS type-stripper using the TypeScript 6 parser.
    
*   [Aedes 1.0.2](https://nodeweekly.com/link/183221/0a4070b383) – Node.js MQTT broker that runs on any stream server.
    
*   [Ky 2.0 Prerelease](https://nodeweekly.com/link/183222/0a4070b383) – Popular HTTP client based on the Fetch API.
    
*   [Got 15.0](https://nodeweekly.com/link/183223/0a4070b383) – Human-friendly, powerful HTTP request library.
    
*   [ESLint Markdown Plugin 8.0](https://nodeweekly.com/link/183224/0a4070b383) – Lint Markdown with ESLint.
    
*   [file-type 22.0](https://nodeweekly.com/link/183225/0a4070b383) – Detect the file type of binary files.

*   [WWWBasic](https://nodeweekly.com/link/183229/0a4070b383) is a curious Google-hosted project to use the [BASIC](https://nodeweekly.com/link/183230/0a4070b383) language in `script` tags or within Node apps. It's had its first update in years (1.0.2) to add ESM support and improve a few things. This seems to be a true labor of love for Googler Brad Nelson.
