[The Roadmap to AdonisJS 7](https://nodeweekly.com/link/170791/0a4070b383 "adonisjs.com") — [Adonis](https://nodeweekly.com/link/170792/0a4070b383) is a popular TypeScript-first 'batteries included' web framework with a rich set of features, and its developers say they’re “shifting gears” and stepping up with more frequent major releases. v7 promises a lot, including Node.js diagnostic channel support, a type-safe URL builder, a new encryption layer, first-class support for notifications and TanStack Query, plus more. You’re encouraged to [give your feedback here.](https://nodeweekly.com/link/170793/0a4070b383)

[Master Full Stack TypeScript with Steve Kinney](https://nodeweekly.com/link/170790/0a4070b383 "frontendmasters.com") — Overcome the challenges of end-to-end type safety in this detailed video course, covering Zod schema validation, API type safety, tRPC, and more.

**IN BRIEF:**

*   [Node v20.19.3 (LTS)](https://nodeweekly.com/link/170795/0a4070b383) has been released with Ed25519 and X25519 promoted to stable in WebCryptoAPI, root certificate and dependency updates, and some V8 backports for fix builds on Xcode.
    
*   Popular alternative server-side JS runtime [Bun v1.2.17](https://nodeweekly.com/link/170796/0a4070b383) supports ahead-of-time bundling in server-side code, and has [even more Node.js compatibility improvements.](https://nodeweekly.com/link/170797/0a4070b383)
    
*   [Vite 7.0 has just been released](https://nodeweekly.com/link/170798/0a4070b383) as we're about to send – we'll be covering it in more depth in [JavaScript Weekly](https://nodeweekly.com/link/170799/0a4070b383) this Friday.
    

[Worker Threads: A Guide to Multithreading in Node](https://nodeweekly.com/link/170800/0a4070b383 "nodesource.com") — Node famously encourages a non-blocking event-driven approach running, largely, on a single thread, but Node’s now-mature [worker threads](https://nodeweekly.com/link/170801/0a4070b383) feature opens the door to multi-threaded apps, should you need it.

Lizz Parody

[Benchmarking Serialization Approaches in Node](https://nodeweekly.com/link/170802/0a4070b383 "adamfaulkner.github.io") — A comparison of several serialization approaches including JSON and binary options like protobuf and msgpack.

Adam Faulkner

[Tips for Making Regular Expressions Easier to Use in JavaScript](https://nodeweekly.com/link/170803/0a4070b383 "2ality.com") — Dr. Axel asks us to imagine if we had to write JavaScript without any whitespace or comments, so why should we have to write regexes that way? He has some tips for making the process more pleasant.

🛠 Code & Tools

[SVGO 4.0: The Node-Powered SVG Optimizer](https://nodeweekly.com/link/170809/0a4070b383 "svgo.dev") — SVG (Scalable Vector Graphics) images often contain a lot of redundant information. You can use SVGO as a CLI tool, Node library, or [even on the Web](https://nodeweekly.com/link/170810/0a4070b383) to optimize SVGs. [Version 4.0](https://nodeweekly.com/link/170811/0a4070b383) is a significant release with changes to its default plugins, public API, and it’s now a dual ESM/CJS package. [GitHub repo.](https://nodeweekly.com/link/170812/0a4070b383)

Kir Belevich

[Hono 4.8: A Cross-Runtime Standards-Oriented Web Framework](https://nodeweekly.com/link/170813/0a4070b383 "github.com") — [Hono](https://nodeweekly.com/link/170814/0a4070b383) is a framework well worth exploring. It’s fast, lightweight, built on Web Standards, and can be used to build apps that work on numerous platforms from Node or Bun to Cloudflare or Fastly. v4.8 adds new route helper functions, improvements to JSX streaming and CORS, a new plugin system for static site generation, and more.

Yusuke Wada and Contributors

[LogTape 1.0.0: Universal Logging for JavaScript Apps](https://nodeweekly.com/link/170815/0a4070b383 "hackers.pub") — Whether it’s in Node, the browser, or edge functions, LogTape is here for you. It’s particularly aimed at library developers who want to add logging in a low hassle way for end users – [learn more here.](https://nodeweekly.com/link/170816/0a4070b383)

Hong Minhee

*   🤖 [OpenAI Node 5.7](https://nodeweekly.com/link/170820/0a4070b383) – The official Node library for OpenAI's APIs.
    
*   [Electron 37.0](https://nodeweekly.com/link/170821/0a4070b383) – The cross-platform desktop app framework.
    
*   [BullMQ 5.56](https://nodeweekly.com/link/170822/0a4070b383) – Reliable, Redis-based distributed queue.
    
*   [Mocha 11.7](https://nodeweekly.com/link/170823/0a4070b383) – Test framework for Node & browsers.

A roundup of some other interesting stories in the broader JavaScript landscape, in case you've missed them:

You can read it online for free, but he's also added [a set of flash cards](https://nodeweekly.com/link/170827/0a4070b383) to help you through the learning process.
