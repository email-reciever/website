[Still Writing Tests Manually? Meticulous AI Is Here](https://react.statuscode.com/link/186520/5d0bbb4a61 "www.meticulous.ai") — Notion, Dropbox, Wiz and LaunchDarkly now use a testing paradigm they can’t work without. Built by former Palantir engineers, Meticulous automatically creates an evolving suite of E2E UI tests, delivering exhaustive coverage with no developer effort.

Meticulous sponsor

[The React Foundation's New Website and...](https://react.statuscode.com/link/186522/5d0bbb4a61 "react.foundation") — The _React Foundation_ [launched in February](https://react.statuscode.com/link/186523/5d0bbb4a61) taking over the ownership and management of React from Meta. All the pieces are continuing to come together with this new site, and **[`facebook/react` now forwards to `react/react`](https://react.statuscode.com/link/186524/5d0bbb4a61) on GitHub** too.

The Linux Foundation

**IN BRIEF:**

*   🤖 The framework and provider-agnostic AI toolkit [TanStack AI is now in beta](https://react.statuscode.com/link/186525/5d0bbb4a61), as is [TanStack Table v9](https://react.statuscode.com/link/186527/5d0bbb4a61).
    
*   Remco Haszing has proposed [returning a disposable from `useEffect` and `useLayoutEffect`](https://react.statuscode.com/link/186529/5d0bbb4a61) instead of a callback function.
    
*   [React Native 0.86.0](https://react.statuscode.com/link/186531/5d0bbb4a61) has been released, as has [React Native Windows v0.83](https://react.statuscode.com/link/186533/5d0bbb4a61).
    

[Stripping Ads from the Kindle's React Native Homepage](https://react.statuscode.com/link/186535/5d0bbb4a61 "sighery.com") — TIL Amazon has been moving the Kindle device UI to React Native. That opens up some interesting Hermes-bytecode-level reverse-engineering opportunities.

Nicholas Tarta

📄 [Building an Active `NavLink` Component in Next.js](https://react.statuscode.com/link/186537/5d0bbb4a61) – A very comprehensive walkthrough. Aurora Scharff

📄 [The Best Loading States Are No Loading States](https://react.statuscode.com/link/186539/5d0bbb4a61) – Skip the skeletons: preload before navigation, then transition when ready. Jenna Smith

📄 [When React Parent Components Need to Know Their Children](https://react.statuscode.com/link/186541/5d0bbb4a61) Jay Freestone

🛠  Code, Tools & Libraries

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/c9w5mttaisyanqcj3n5c.jpg)](https://react.statuscode.com/link/186542/5d0bbb4a61)

📊 [visx 4.0: Airbnb's Visualization Primitives for React](https://react.statuscode.com/link/186542/5d0bbb4a61 "visx.airbnb.tech") — _“If you know React, you can make visualizations.”_ The long-standing visualization component suite is back with full React 19 support. You can [see what all the components do here](https://react.statuscode.com/link/186543/5d0bbb4a61), and there’s an [upgrade guide](https://react.statuscode.com/link/186544/5d0bbb4a61) for existing v3 users.

Airbnb

[The Best Stack for a Money-Making Mobile App in 2026](https://react.statuscode.com/link/186545/5d0bbb4a61 "try.expo.dev") — Real monetization data, the mobile stack that actually ships, and why React Native apps drive more revenue than native.

Expo sponsor

📃 [PDFSlick 4.0: View and Interact with PDF Documents](https://react.statuscode.com/link/186546/5d0bbb4a61 "pdfslick.dev") — A full-featured PDF viewer [with React bindings](https://react.statuscode.com/link/186547/5d0bbb4a61). Built on PDF.js, it handles everything from simple viewing to annotations and juggling multiple large documents. [Demo.](https://react.statuscode.com/link/186548/5d0bbb4a61)

Vancho Stojkov

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/bokip5akj9lnpb2y87a5.jpg)](https://react.statuscode.com/link/186549/5d0bbb4a61)

*   [bezier-easing-editor 1.1](https://react.statuscode.com/link/186550/5d0bbb4a61) _(above)_ – React component for visually editing Bézier curves. [Live demo.](https://react.statuscode.com/link/186549/5d0bbb4a61)
    
*   📺 [React Lite YouTube Embed 3.6](https://react.statuscode.com/link/186551/5d0bbb4a61) – A tiny, privacy-friendly way to embed YouTube videos without a performance hit.
    
*   [react-native-network-logger 3.0](https://react.statuscode.com/link/186552/5d0bbb4a61) – HTTP traffic monitor/request logger for React Native with in-app viewer for iOS and Android.
    
*   [react-native-compressor 2.0](https://react.statuscode.com/link/186553/5d0bbb4a61) – Compress images, videos and audio prior to upload, like many social apps do.
    
*   [React Native Testing Library 14.0](https://react.statuscode.com/link/186554/5d0bbb4a61) – Note that React 19+ and React Native 0.78+ are now required.
    
*   [react-redux-firebase 4.0](https://react.statuscode.com/link/186555/5d0bbb4a61) – Redux bindings for Firebase. Now supports React 19.
    
*   [MUI (Material UI) 9.1.0](https://react.statuscode.com/link/186556/5d0bbb4a61) – The component suite adds support for `prefers-reduced-motion`.
    

📰 Classifieds

📱 [Clerk now ships prebuilt Org management for iOS and Android](https://react.statuscode.com/link/186557/5d0bbb4a61): account switching, member management, and invite flows — all native.

📢  Elsewhere in the ecosystem

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/ik3o5cphjgwepp8vokto.jpg)](https://react.statuscode.com/link/186558/5d0bbb4a61)

*   😆 [performative-ui](https://react.statuscode.com/link/186558/5d0bbb4a61) is a fun set of React components to _'signal how oversubscribed your funding round is'_ with all the AI company clichés, like [ASCII hero art](https://react.statuscode.com/link/186559/5d0bbb4a61) and [node graph backgrounds](https://react.statuscode.com/link/186560/5d0bbb4a61).
    
*   Alistair Davidson writes about [how building an HTML-first site doubled user submissions overnight](https://react.statuscode.com/link/186561/5d0bbb4a61) after a failed and expensive attempt with React.
    
*   At WWDC this week, Apple [released the beta of Safari 27](https://react.statuscode.com/link/186562/5d0bbb4a61) and it's packed with features, including customizable `&lt;select>` elements, scroll anchoring, and an all-new standards-compliant [ESM loader](https://react.statuscode.com/link/186563/5d0bbb4a61).
    
*   Get ready for your Electron apps to become faster! [Electron 43 is now in beta](https://react.statuscode.com/link/186564/5d0bbb4a61) with the main process booting from an embedded Node.js startup snapshot and bundles cached as V8 bytecode.
