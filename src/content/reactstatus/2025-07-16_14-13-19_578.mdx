This will round out your thinking about React if you weren’t using it from the earliest days.

Robin Wieruch sponsor

[Announcing Node-API Support for React Native](https://react.statuscode.com/link/171775/5d0bbb4a61 "www.callstack.com") — A huge development for React Native. By bringing Node’s native module system into React Native, many doors are opened for code-sharing between platforms, prebuilding native modules for faster build times, and using existing Node-API compatible modules in React Native for the first time.

Hansen, Pasinski, et al. (Callstack)

🤖 Callstack has another announcement, with [a new package that brings Apple's on-device foundation models to React Native.](https://react.statuscode.com/link/171776/5d0bbb4a61)

**IN BRIEF:**

*   🐝 [Wasp](https://react.statuscode.com/link/171777/5d0bbb4a61) is a popular Ruby on Rails-like framework for React, Node.js and Prisma, and it now has [a public development roadmap.](https://react.statuscode.com/link/171778/5d0bbb4a61)
    
*   You may know Lee Robinson from his years building up Vercel's devrel program and growing the Next.js community. Well, he's just left Vercel and [shares five things he learned from five years at Vercel.](https://react.statuscode.com/link/171779/5d0bbb4a61)
    
*   Reddit's `/r/reactjs` got to [discussing Remix vs Next.js](https://react.statuscode.com/link/171780/5d0bbb4a61) and which one you should choose.
    

[Next.js 15.4 Released (and What's Coming in Next.js 16)](https://react.statuscode.com/link/171781/5d0bbb4a61 "nextjs.org") — A relatively small release for Next, but with updates to performance, stability, and Turbopack compatibility, and a good summary of what’s coming next in Next.js 16.

Jimmy Lai and Zack Tanner

[The World’s Longest React Hooks Migration?](https://react.statuscode.com/link/171782/5d0bbb4a61 "craft.faire.com") — The three year tale of how a team transitioned from class components and MobX classes to functional components and hooks in a large React app.

Chris Krogh

[How to Build an AI Coding Rules App with Lovable](https://react.statuscode.com/link/171783/5d0bbb4a61 "go.clerk.com") — Guide AI to generate a secure, full-stack app with minimal prompts—learn how to turn ideas into working software fast.

Clerk sponsor

🛠  Code, Tools & Libraries

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/lyls7dpzj0sgngjvwdh3.jpg)](https://react.statuscode.com/link/171788/5d0bbb4a61)

[react-easy-crop: A Component for Interactive Image Cropping](https://react.statuscode.com/link/171788/5d0bbb4a61 "valentinh.github.io") — Supports any image format (and even videos) along with dragging, zooming, and rotations. [GitHub repo.](https://react.statuscode.com/link/171789/5d0bbb4a61)

Valentin Hervieu

📺 [ReactPlayer 3.2: A Component for Playing Media from URLs](https://react.statuscode.com/link/171790/5d0bbb4a61 "github.com") — As well as standard videos, it can play HLS streams, DASH streams, YouTube videos, Vimeo videos, and more. There’s [a complete online demo](https://react.statuscode.com/link/171791/5d0bbb4a61) showing off its flexibility.

Pete Cook and Mux

[Hyper Fetch: A 'Turbocharged' Fetch Library for Working with Remote APIs](https://react.statuscode.com/link/171792/5d0bbb4a61 "hyperfetch.bettertyped.com") — An Axios and TanStack Query-inspired type-safe data-fetching framework for browser and server environments, with request lifecycle management, real-time communication, progress tracking, and codegen for Swagger/OpenAPI. [GitHub repo.](https://react.statuscode.com/link/171793/5d0bbb4a61)

Maciej Pyrc et al.

🌧️ [Animated WebGL-Powered Weather Effects](https://react.statuscode.com/link/171794/5d0bbb4a61 "github.com") — A quick, fun one with [a demo](https://react.statuscode.com/link/171795/5d0bbb4a61) of its rain, snow, and fog effects.

Mate Rauscher

*   🖼️ [react-medium-image-zoom 5.3](https://react.statuscode.com/link/171796/5d0bbb4a61) – Medium​.com inspired image component. ([Numerous live examples.](https://react.statuscode.com/link/171797/5d0bbb4a61))
    
*   [Wasp 0.17](https://react.statuscode.com/link/171798/5d0bbb4a61) – [Wasp](https://react.statuscode.com/link/171799/5d0bbb4a61) is a Rails-like framework using Node, React & Prisma.
    
*   ♟︎ [React Chessboard 5.2](https://react.statuscode.com/link/171800/5d0bbb4a61) – Render chess boards. ([Examples.](https://react.statuscode.com/link/171801/5d0bbb4a61))
    
*   [Tinybase 6.4](https://react.statuscode.com/link/171802/5d0bbb4a61) – Powerful reactive data store for local‑first apps.
    
*   [MUI X 8.8](https://react.statuscode.com/link/171803/5d0bbb4a61) – Popular React component suite.
    

📢  Elsewhere in JavaScript

A roundup of some other interesting stories in the broader JavaScript landscape, in case you've missed them:

*   [New versions of all maintained Node.js release lines have just been released](https://react.statuscode.com/link/171804/5d0bbb4a61) including Node v20.19.4, v22.17.1, and v24.4.1.
    
*   📅 How well do you know JavaScript's `Date` class and how date parsing works in JavaScript? Find out with [this head-scratching quiz.](https://react.statuscode.com/link/171807/5d0bbb4a61)
    
*   Last week, [Vercel announced it had acquired NuxtLabs](https://react.statuscode.com/link/171805/5d0bbb4a61), the stewards of Nitro and Nuxt (think Next.js but for Vue.js), meaning Next.js, Turborepo, Svelte, and now the Vue.js-powered Nuxt are within its sphere. [Nuxt 4.0](https://react.statuscode.com/link/171806/5d0bbb4a61) has also just been released.
    
*   It's written in Go, but [fx](https://react.statuscode.com/link/171808/5d0bbb4a61) is a fantastic JSON viewer and processor for the terminal – handy if you're working with JSON at scale. [v37.0](https://react.statuscode.com/link/171809/5d0bbb4a61) brings Vim-style line jumping and JSON parsing enhancements.
