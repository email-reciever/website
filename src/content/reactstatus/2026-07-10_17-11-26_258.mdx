[Reverse Engineering ChatGPT Web: How OpenAI Built for a Billion Users](https://react.statuscode.com/link/187794/5d0bbb4a61 "performance.dev") — The author of the popular [Linear](https://react.statuscode.com/link/187795/5d0bbb4a61) performance breakdown spent days digging through ChatGPT's page source, bundles, and requests. React Router 7, TanStack Query, Radix, and ProseMirror all show up, with every non-essential script deferred until you can type.

Dennis Brotzky

[_Base UI_ is Now the Default in `shadcn/ui`](https://react.statuscode.com/link/187796/5d0bbb4a61 "ui.shadcn.com") — New [shadcn/ui](https://react.statuscode.com/link/187797/5d0bbb4a61) projects now build on [Base UI](https://react.statuscode.com/link/187798/5d0bbb4a61) rather than [Radix](https://react.statuscode.com/link/187799/5d0bbb4a61). Radix _isn't_ deprecated and every component ships for both, but _"Starting something new? We recommend Base UI."_

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/162511c4.png)](https://react.statuscode.com/link/187793/5d0bbb4a61)

[Ship Translations Without Redeploying Your App](https://react.statuscode.com/link/187793/5d0bbb4a61 "www.locize.com") — The cloud translation backend made by the i18next team. saveMissing, in-context editing, AI translation, and instant CDN updates. No redeploys for content changes. Free tier, Swiss-made.

Locize sponsor

[What Triggers Suspense Boundaries in React?](https://react.statuscode.com/link/187800/5d0bbb4a61 "react.dev") — The official React docs have been updated with a list of what can activate a Suspense boundary, with a live example for each trigger. Useful for understanding exactly when fallback UI appears in real apps. Aurora [explains further on X](https://react.statuscode.com/link/187801/5d0bbb4a61).

Aurora Scharff / React.dev Docs

**IN BRIEF:**

*   [TypeScript 7.0](https://react.statuscode.com/link/187802/5d0bbb4a61), the 10x faster Go-powered version of the TypeScript compiler, has been released. With no programmatic API yet, however, [6.0 is still recommended](https://react.statuscode.com/link/187802/5d0bbb4a61) for many frontend frameworks.
    
*   [Vercel has acquired](https://react.statuscode.com/link/187803/5d0bbb4a61) the team behind [Better Auth](https://react.statuscode.com/link/187804/5d0bbb4a61), the popular authentication library. Flavio Copes [has a quick intro post](https://react.statuscode.com/link/187805/5d0bbb4a61) if you're not familiar with it.
    
*   On Reddit, `/r/reactjs` discussed [React patterns they stopped using after working on larger projects](https://react.statuscode.com/link/187806/5d0bbb4a61). `useEffect` came up a _lot_.
    
*   The [React Navigation](https://react.statuscode.com/link/187807/5d0bbb4a61) team shares [a progress update on React Navigation 8.0](https://react.statuscode.com/link/187808/5d0bbb4a61) including Suspense-enabled navigation.
    
*   There's [an official RFC for WebSocket support in Next.js Route Handlers](https://react.statuscode.com/link/187809/5d0bbb4a61).
    

[React Forms Done Right](https://react.statuscode.com/link/187810/5d0bbb4a61 "upskills.dev") — An interactive dive into web forms, starting with zero-JS HTML forms and moving through React 19 native form actions to using [TanStack Form](https://react.statuscode.com/link/187811/5d0bbb4a61) with Zod. Complete with several demos.
