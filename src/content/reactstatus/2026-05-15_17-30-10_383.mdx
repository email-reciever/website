[From Latency to Instant: Improving _GitHub Issues_ Navigation Performance](https://react.statuscode.com/link/185269/5d0bbb4a61 "github.blog") — GitHub rebuilt issue navigation around a local-first, stale-while-revalidate model: render from an IndexedDB cache, revalidate in the background. ~70% of React navigations are now instant & P10 down from 600ms to 70ms. A deep account of the caching & prefetch design on GitHub's React/Rails stack.

Alexander Lelidis (GitHub)

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/0dd4c496.png)](https://react.statuscode.com/link/185264/5d0bbb4a61)

[Microfrontends, Monoliths, & Monorepos](https://react.statuscode.com/link/185264/5d0bbb4a61 "frontendmasters.com") — Discover how to architect large, successful frontend systems in this detailed video course, hosted by Steve Kinney. Learn best practices for maintaining complex codebases while shipping high-quality software.

▶  [Why TanStack Start Exists: Tanner Linsley on Competing with Next.js](https://react.statuscode.com/link/185268/5d0bbb4a61 "www.youtube.com") — A candid interview with Tanner Linsley covering a wide variety of topics like TanStack’s business model, Start vs Next.js, and TanStack’s take on being framework agnostic while still focusing on React.

Nuno Maduro

🔒 [Hardening TanStack After the npm Compromise](https://react.statuscode.com/link/185265/5d0bbb4a61 "tanstack.com") — This week, [an attacker published](https://react.statuscode.com/link/185266/5d0bbb4a61) malicious versions of 42 TanStack packages using a novel approach (it affected [170+ other packages too](https://react.statuscode.com/link/185267/5d0bbb4a61)). Here’s what the TanStack team is doing to prevent such issues going forward.

The TanStack Team

[A Technical Deep Dive Into the New Raycast](https://react.statuscode.com/link/185270/5d0bbb4a61 "www.raycast.com") — [Raycast](https://react.statuscode.com/link/185271/5d0bbb4a61) is a feature-rich macOS launcher app (imagine Spotlight but 10x) which has long had a React-based extension ecosystem, but now its new v2.0 app has a React frontend too.

Nikolaev and Mann (Raycast)

[Your Agent Burned Through the Free Tier by Lunch](https://react.statuscode.com/link/185272/5d0bbb4a61 "ghost.build") — _ghost_ gives your agent unlimited Postgres. Fork per task, delete after. No project limits. 1TB free. [Try for free](https://react.statuscode.com/link/185272/5d0bbb4a61).

ghost sponsor

📄 [Security in React Applications](https://react.statuscode.com/link/185273/5d0bbb4a61) – High-level tips and techniques for securing React apps. Aurora Scharff

📄 [RSC Server Functions Are Not An API Boundary](https://react.statuscode.com/link/185274/5d0bbb4a61) Long Ho

📄 [The Hidden Cost of &lt;Activity>](https://react.statuscode.com/link/185275/5d0bbb4a61) Peter Piekarczyk

🛠  Code, Tools & Libraries

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/tze0hinzt0gj9fqfqi3w.jpg)](https://react.statuscode.com/link/185276/5d0bbb4a61)

[MDXEditor 4.0: A Rich Markdown Editor Component](https://react.statuscode.com/link/185276/5d0bbb4a61 "mdxeditor.dev") — A [Lexical](https://react.statuscode.com/link/185277/5d0bbb4a61)\-powered Markdown editor for React apps with a wide array of features like code editing, link/image insertion, and table support. [The live demo](https://react.statuscode.com/link/185278/5d0bbb4a61) shows off what it can do with all the features enabled.

Petyo Ivanov

[Fate 1.0: A Modern Data Framework for React](https://react.statuscode.com/link/185304/5d0bbb4a61 "fate.technology") — A new data framework from former Jest lead and ex-Meta engineer Christoph Nakazawa. Brings normalized caching, view co-location, and single-root-request composition on top of tRPC or native HTTP, with Prisma and now Drizzle support on the server. 1.0 adds live views over SSE, garbage collection, and a Vite plugin.

Christoph Nakazawa

🤖 [React Review: Audit Your Codebase for Bad React Code](https://react.statuscode.com/link/185279/5d0bbb4a61 "react.review") — From the creator of [React Doctor](https://react.statuscode.com/link/185280/5d0bbb4a61) comes a new GitHub App (_powered_ by React Doctor) that fetches both a pull request branch and base branch and reviews and reports on issues introduced in the PR.

Aiden Bai et al.

[Orval: Generate Type-Safe Clients from OpenAPI/Swagger Specs](https://react.statuscode.com/link/185282/5d0bbb4a61 "github.com") — Given a valid OpenAPI v3 or Swagger v2 spec, generate models, requests, hooks, and mocks for React, Vue, Svelte, and Solid apps and more.

Victor Bury

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/r23bo3hkjgsxml3auhl2.jpg)](https://react.statuscode.com/link/185283/5d0bbb4a61)

*   [Mantine 9.2](https://react.statuscode.com/link/185283/5d0bbb4a61) – The popular component suite adds [TreeSelect](https://react.statuscode.com/link/185284/5d0bbb4a61) _(above)_ for selecting hierarchical tree data, the dropdown-free [InlineDateTimePicker](https://react.statuscode.com/link/185285/5d0bbb4a61), [RollingNumber](https://react.statuscode.com/link/185286/5d0bbb4a61) to animate value changes, a new `use-drag` hook, and more.
    
*   ⭐ [Waku 1.0 Beta](https://react.statuscode.com/link/185287/5d0bbb4a61) – The minimal React Server Components framework built atop Vite and Hono.
    
*   [whisper.rn 0.6](https://react.statuscode.com/link/185288/5d0bbb4a61) – React Native bindings to `whisper.cpp` for fast, on-device speech transcription.
    
*   [Ant Design 6.4.0](https://react.statuscode.com/link/185289/5d0bbb4a61) – Enterprise-class UI design language & component library.
    
*   [react-native-view-shot 5.1](https://react.statuscode.com/link/185290/5d0bbb4a61) – Capture a React Native view to an image.
    
*   [react-qr-scanner 2.6](https://react.statuscode.com/link/185291/5d0bbb4a61) – Scan QR codes in React apps.
    
*   [styled-components v7 Alpha](https://react.statuscode.com/link/185292/5d0bbb4a61)
    

📰 Classifieds

Flaky tests slowing down dev? [Meticulous](https://react.statuscode.com/link/185293/5d0bbb4a61) gives engineers confidence to ship faster by autonomously testing every edge case of your web app.

* * *

🔐 [Clerk Directory Sync is now GA](https://react.statuscode.com/link/185294/5d0bbb4a61). Changes in Okta or Entra ID sync automatically to your app — no manual account management. Custom attribute mapping and IdP group-to-role assignment included.

* * *

[Trigger.dev](https://react.statuscode.com/link/185295/5d0bbb4a61) handles queues, retries, and long-running tasks so you can build production-ready agents and TypeScript workflows reliably at scale.

📢  Elsewhere in the ecosystem

*   🤖 Redux maintainer Mark Erikson goes deep with [his personal thoughts and experiences on the use of AI](https://react.statuscode.com/link/185296/5d0bbb4a61) in software development.
    
*   Daishi Kato shares [a brief update on Jotai](https://react.statuscode.com/link/185297/5d0bbb4a61) ([v2.20.0](https://react.statuscode.com/link/185298/5d0bbb4a61) was released last week) and says he's ready to start thinking about Jotai v3.
    
*   🤖 [TanStack AI](https://react.statuscode.com/link/185299/5d0bbb4a61), the cross-provider AI SDK, [can now _stream_ typed structured output](https://react.statuscode.com/link/185300/5d0bbb4a61) with a Zod schema.
    
*   [zero-native](https://react.statuscode.com/link/185301/5d0bbb4a61) is Vercel's entry into the ecosystem of frameworks for building native JS desktop apps. It sits atop a Zig core and lets you pick between the system WebView or Chromium. There are [examples](https://react.statuscode.com/link/185302/5d0bbb4a61) covering how to build vanilla, React, Svelte, and Vue apps on top of it.
    
*   Luciano Mammino looks at [what's new in Node 26.0](https://react.statuscode.com/link/185303/5d0bbb4a61).
