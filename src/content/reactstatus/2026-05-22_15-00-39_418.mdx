Meticulous AI Is Here](https://react.statuscode.com/link/185546/5d0bbb4a61 "www.meticulous.ai") — Notion, Dropbox, Wiz and LaunchDarkly now use a testing paradigm they can’t work without. **IN BRIEF:**

*   [Storybook 10.4](https://react.statuscode.com/link/185549/5d0bbb4a61) – the frontend component workshop – has added first-class TanStack React support, cleaner React Native setup, and the ability for AI agents to automatically set up Storybook in complex apps.
    
*   🗣️ Reddit's `/r/reactjs` discussed [why people are moving from Next.js to TanStack Start.](https://react.statuscode.com/link/185550/5d0bbb4a61)
    
*   [Redux Toolkit v2.12.0](https://react.statuscode.com/link/185551/5d0bbb4a61) and [React Redux 9.3.0](https://react.statuscode.com/link/185552/5d0bbb4a61) have been released.
    

🤖 [Migrating an Express App to Next.js Using Agents](https://react.statuscode.com/link/185553/5d0bbb4a61 "javascript.plainenglish.io") — James (who works on [Antigravity](https://react.statuscode.com/link/185554/5d0bbb4a61) at Google) demonstrates how to modernize a legacy Express.js app to Next.js App Router, TypeScript, MongoDB, Zod, and shadcn/ui. It’s all orchestrated using [these skills](https://react.statuscode.com/link/185555/5d0bbb4a61) which could be used with other, non-Google agents too.

James O'Reilly (Google)

💡 Google [has deprecated Gemini CLI](https://react.statuscode.com/link/185556/5d0bbb4a61), replacing it with the new [Antigravity CLI](https://react.statuscode.com/link/185557/5d0bbb4a61).

🤖 [Mark Erikson's Agent Setup, Workflow, and Tools](https://react.statuscode.com/link/185558/5d0bbb4a61 "blog.isquaredsoftware.com") — Mark, well known for maintaining Redux and creating Redux Toolkit, digs deep into his daily dev workflow, including his use of [OpenCode](https://react.statuscode.com/link/185559/5d0bbb4a61) and how he manages his workflow.

Mark Erikson

📄 [Structural Sharing, `selectAtom`, and Why Your Jotai Atoms Re-Render Too Much](https://react.statuscode.com/link/185560/5d0bbb4a61) Peter Piekarczyk

🛠  Code, Tools & Libraries

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/dmd0wpeuxhfnkw3l4la1.jpg)](https://react.statuscode.com/link/185564/5d0bbb4a61)

[GL-React v6: Create WebGL Shaders from React](https://react.statuscode.com/link/185564/5d0bbb4a61 "gre.github.io") — Provides a straightforward abstraction for creating WebGL shaders with React and JSX. [Gaëtan](https://react.statuscode.com/link/185565/5d0bbb4a61) is an experienced generative and shader artist and there are [lots of examples](https://react.statuscode.com/link/185566/5d0bbb4a61) complete with code to enjoy here.

Gaëtan Renaudeau

[From React Web to a Real iOS App in One Week](https://react.statuscode.com/link/185567/5d0bbb4a61 "try.expo.dev") — A web dev's honest take on what transferred to native, what didn't, and how Expo + Claude Code + Skills closed the gap.

Expo sponsor

💡 [Expo SDK 56](https://react.statuscode.com/link/185568/5d0bbb4a61) has also just been released with Expo UI now considered ready for production.

[Hot Updater: Self-Hosted Over-the-Air Updates for React Native](https://react.statuscode.com/link/185569/5d0bbb4a61 "hot-updater.dev") — A way to deploy updates to apps without app store submissions. The latest update adds [bundle diffing](https://react.statuscode.com/link/185570/5d0bbb4a61) for smaller, more efficient updates.

Sungyu Kang

[Relay 21.0: Meta’s Data-Fetching Framework for React](https://react.statuscode.com/link/185571/5d0bbb4a61 "github.com") — [Relay](https://react.statuscode.com/link/185572/5d0bbb4a61), Meta’s declarative GraphQL-based data fetching framework used in Facebook itself, adds first-party TypeScript support, new error handling capabilities, and **experimental React Server Components support.**

Facebook / Meta

🗓️ [SVAR Calendar: A Calendar Component for React, Svelte and Vue](https://react.statuscode.com/link/185573/5d0bbb4a61 "svar.dev") — A flexible calendar component with a MIT-licensed core and extended commercial version. [Here are some demos](https://react.statuscode.com/link/185586/5d0bbb4a61) of the React version.

XB Software Sp. z o.o.

💡 [Schedule-X](https://react.statuscode.com/link/185575/5d0bbb4a61) is another great option in this space and [v4.6](https://react.statuscode.com/link/185576/5d0bbb4a61) just landed.

*   [Rockpack 8.0.0](https://react.statuscode.com/link/185577/5d0bbb4a61) – A tool for quickly setting up a React app with full support for Server-Side Rendering, bundling, linting, and testing. v8.0 now supports ESLint 10, TypeScript 6, and Tailwind CSS v4.
    
*   [React Native Zip Archive 8.0](https://react.statuscode.com/link/185578/5d0bbb4a61) – Zip archive functions for React Native apps.
    

📰 Classifieds

🔑 [Clerk API Keys are GA](https://react.statuscode.com/link/185579/5d0bbb4a61). Users can generate credentials that delegate API access on their behalf. Verify server-side, revoke instantly.

* * *

[Trigger.dev](https://react.statuscode.com/link/185580/5d0bbb4a61) handles queues, retries, and long-running tasks so you can build production-ready agents and TypeScript workflows reliably at scale.

* * *

Build AR/VR apps in React Native — no Unity, no new language. [ReactVision Studio](https://react.statuscode.com/link/185581/5d0bbb4a61) has a visual scene editor, prompt‑to‑3D asset gen, Meta Quest support, and works with Expo. [Try free](https://react.statuscode.com/link/185581/5d0bbb4a61).

📢  Elsewhere in the ecosystem

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/h9h6kjfbbchyi0q7decy.jpg)](https://react.statuscode.com/link/185582/5d0bbb4a61)

*   🤖 Google unveiled [Modern Web Guidance](https://react.statuscode.com/link/185582/5d0bbb4a61) at _I/O_ this week. It's a set of expert-vetted skills to guide coding agents to build accessible and performant web experiences, e.g. `"Set up my app to begin preloading pages when users hover over important links."`
    
*   [npq](https://react.statuscode.com/link/185583/5d0bbb4a61) is a tool that makes `npm install` safer. It sits in front of `npm` and audits packages prior to installation, including warning about pre/post-install scripts.
    
*   Node.js has a new official [codemod for migrating from Axios to `fetch`](https://react.statuscode.com/link/185584/5d0bbb4a61).
