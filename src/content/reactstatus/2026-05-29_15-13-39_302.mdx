[Pairing TanStack Router with TanStack Query](https://react.statuscode.com/link/185889/5d0bbb4a61 "tkdodo.eu") — Dominik continues [his series on TanStack Router](https://react.statuscode.com/link/185890/5d0bbb4a61), this time pairing it with [TanStack Query](https://react.statuscode.com/link/185891/5d0bbb4a61). The router cache is per-route while Query’s is global, making Query a better fit for data shared across routes. He covers wiring the `QueryClient` into router context, disabling the router’s own caching so one library owns the data layer, and starting fetches in loaders that `useSuspenseQuery` reads from.

Dominik Dorfmeister

💡 If you don't use it yet, [The Beauty of TanStack Router](https://react.statuscode.com/link/185890/5d0bbb4a61) might win you over.

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/7d59b09e.png)](https://react.statuscode.com/link/185888/5d0bbb4a61)

[Still Writing Tests Manually? Meticulous AI Is Here](https://react.statuscode.com/link/185888/5d0bbb4a61 "www.meticulous.ai") — Notion, Dropbox, Wiz and LaunchDarkly now use a testing paradigm they can’t work without. Built by former Palantir engineers, Meticulous automatically creates an evolving suite of E2E UI tests, delivering exhaustive coverage with no developer effort.

Meticulous sponsor

[As React Turns 13, 'Does Anybody Actually Like React?'](https://react.statuscode.com/link/185892/5d0bbb4a61 "jsx.lol") — First released on May 29, 2013, React is only just entering its teenage era, but complaints about React’s omnipresence have been a trope for years and David Bushell has been keeping receipts!

David Bushell

🗣️ This item kicked off [an extensive discussion on Hacker News](https://react.statuscode.com/link/185921/5d0bbb4a61).

**IN BRIEF:**

*   [React Router 7.16.0](https://react.statuscode.com/link/185893/5d0bbb4a61) lands as the on-ramp to v8, stabilizing the [trailing-slash data-request](https://react.statuscode.com/link/185894/5d0bbb4a61) flag and warning about every v8 flag you haven't enabled yet.
    
*   🇳🇱 [React Summit](https://react.statuscode.com/link/185895/5d0bbb4a61) is taking place this June 12 in Amsterdam (the day after [JSNation](https://react.statuscode.com/link/185896/5d0bbb4a61)). Both can be attended remotely too.
    
*   🇺🇸 The annual [Chain React](https://react.statuscode.com/link/185897/5d0bbb4a61) React Native conference is this July 30-31 in Portland, Oregon.
    

[React at 60 FPS](https://react.statuscode.com/link/185899/5d0bbb4a61 "www.readwriterachel.com") — A slide deck packed with tips for squeezing performance out of React when animating data, without reaching for an animation library — instead leaning on Canvas, `requestAnimationFrame`, WebSockets, and `React.memo`.

Rachel Kaufman

▶  [Is TanStack Start's Deferred Hydration Revolutionary?](https://react.statuscode.com/link/185900/5d0bbb4a61 "www.youtube.com") — Jack examines TanStack Start’s deferred hydration, a new feature that delays hydrating below-the-fold components until they’re needed, noting it’s syntactic sugar over capabilities React already had, though he reckons it’s worth it.

Jack Herrington

💡 TanStack's own docs also offer [a full guide to deferred hydration](https://react.statuscode.com/link/185901/5d0bbb4a61).

[Expo UI Is Stable. Real SwiftUI and Compose from JS](https://react.statuscode.com/link/185902/5d0bbb4a61 "try.expo.dev") — One import. SwiftUI on iOS, Jetpack Compose on Android. Plus native drop-in replacements for 7 community packages.

[Spoiled 0.5: A Realistic 'Spoiler' Component for React](https://react.statuscode.com/link/185912/5d0bbb4a61 "spoiled.vercel.app") — Displays a cloud of particles over content that should stay obscured until revealed. Alexey Taktarov

*   [React Native Reanimated 4.4](https://react.statuscode.com/link/185915/5d0bbb4a61) – Easily build smooth animations on React Native. v4.4 has a new CSS animation engine on iOS that runs animations through Core Animation instead of a JS-driven update loop.
    
*   👉 [React Native Gesture Handler 3.0](https://react.statuscode.com/link/185916/5d0bbb4a61) – Exposes platform-native touch/gesture systems to React Native. Now rebuilt for the New Architecture and with a new `Touchable` component.
    
*   ↔ [React Archer 5.0](https://react.statuscode.com/link/185917/5d0bbb4a61) – Draw arrows between DOM elements in React. v5.0 now supports React 19.
    
*   [Sugar High 1.2](https://react.statuscode.com/link/185918/5d0bbb4a61) – Super-light (1KB) syntax highlighter for JavaScript and JSX.
    
*   [React Spring 10.1](https://react.statuscode.com/link/185919/5d0bbb4a61) – A mild bump for the wildly popular UI animation library.
    
*   [MUI X 9.3](https://react.statuscode.com/link/185920/5d0bbb4a61) – The popular component suite.
