[React Native's _New Architecture_ is Here](https://react.statuscode.com/link/161612/5d0bbb4a61 "reactnative.dev") — With the [release of React Native 0.76](https://react.statuscode.com/link/161613/5d0bbb4a61), the so-called ‘New Architecture’ becomes the default. This opens up numerous modern React features like `useLayoutEffect` and Suspense, as well as better ways to work with native interfaces without a bridge (in short, working with the native runtime is faster and more direct).

The React Team

[React Compiler Beta Release](https://react.statuscode.com/link/161614/5d0bbb4a61 "react.dev") — Five months ago we got our first official introduction to [React Compiler](https://react.statuscode.com/link/161615/5d0bbb4a61), a new approach to improving React app performance by optimizing code at build time. It’s now a step closer, and React 17+ users and library maintainers are encouraged to give it a spin.

You'll learn how to best use Lighthouse, WebPageTest and other tools in order to make your site lightning.

[React Native 的 *新架构* 就在这里](https://react.statuscode.com/link/161612/5d0bbb4a61 "reactnative.dev") — 随着 \[React Native 0.76 版本]\(https://react.statuscode .com/link/161613/5d0bbb4a61），所谓的“新架构”成为默认架构。这开启了许多现代 React 功能，如“useLayoutEffect”和 Suspense，以及无需桥梁即可使用本机界面的更好方法（简而言之，使用本机运行时更快、更直接）。

反应团队

[React Compiler Beta 版本](https://react.statuscode.com/link/161614/5d0bbb4a61 "react.dev") — 五个月前，我们首次正式介绍了 [React Compiler](https://react.dev)。 statuscode.com/link/161615/5d0bbb4a61），一种通过在构建时优化代码来提高 React 应用程序性能的新方法。现在又近了一步，我们鼓励 React 17+ 用户和库维护者尝试一下。

劳伦·谭

[![](https://copm.s3.amazonaws.com/0ef424ba.png)](https://react.statuscode.com/link/161611/5d0bbb4a61)

[掌握性能并提高网站速度](https://react.statuscode.com/link/161611/5d0bbb4a61 "frontendmasters.com") — 这个详细的视频课程分享了 Web 性能的基础知识，涵盖了关键的核心 Web Vitals 指标，例如如累积布局偏移 (CLS) 和与下一个绘制的交互 (INP)。您将学习如何最好地使用 Lighthouse、WebPageTest 和其他工具，让您的网站变得闪电般。

前端大师赞助商
