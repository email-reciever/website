Robin Wieruch

Matt Carroll and Ricky Hanlon

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/09c4bd93.png)](https://react.statuscode.com/link/166163/5d0bbb4a61)

[SurveyJS: A Powerful React Form Builder that Works with Any Backend](https://react.statuscode.com/link/166163/5d0bbb4a61) — Integrate the SurveyJS form-building UI component into your React app and design JSON-powered forms of any complexity without manual coding. Fully customizable, drag-and-drop, compatible with React 19 and any backend. Perfect for data-heavy applications. Start with a free demo.

SurveyJS sponsor

📊 [The State of React Native 2024](https://react.statuscode.com/link/166165/5d0bbb4a61 "results.stateofreactnative.com") — In the last issue we linked to the [results of the _State of React 2024_ survey](https://react.statuscode.com/link/166166/5d0bbb4a61), now it’s React Native’s turn. It gives a good look at what React Native developers are using in modern apps, particularly if you’re not familiar with the space, and it’s interesting to see [how widely popular Expo now is.](https://react.statuscode.com/link/166167/5d0bbb4a61)

Software Mansion

**IN BRIEF:**

*   The tRPC project has unveiled [a new integration with TanStack React Query.](https://react.statuscode.com/link/166171/5d0bbb4a61)
    

[React Native 0.78 Released](https://react.statuscode.com/link/166168/5d0bbb4a61 "reactnative.dev") — A big release for React Native as it ships React 19 support (which [requires some tweaks when upgrading](https://react.statuscode.com/link/166169/5d0bbb4a61)) and some more minor tweaks.

Novak, Cucci, Gupta and Cipolleschi (Meta)

[Using F# to Build React Apps: Transpiling to JS with Fable](https://react.statuscode.com/link/166172/5d0bbb4a61 "www.compositional-it.com") — The first in a promised series of posts on using Microsoft’s functional language F# to build React apps. Step one is to turn your JavaScript into F# by using [Fable.](https://react.statuscode.com/link/166173/5d0bbb4a61)

Jaz Thomson

💡 Part two of the series covers [using Feliz to write React components in F#](https://react.statuscode.com/link/166208/5d0bbb4a61) and there are [further posts here.](https://react.statuscode.com/link/166209/5d0bbb4a61)

[Find. Fix. Test: An Intro to App Monitoring with Sentry](https://react.statuscode.com/link/166174/5d0bbb4a61 "sentry.io") — Code-level visibility, from pre- to post-release, lets devs find and fix errors and slowdowns and deploy with confidence.

Sentry sponsor

📺 [React Router 7.2 Just Added a 'Game Changing' Feature](https://react.statuscode.com/link/166175/5d0bbb4a61) – It’s a new type-safe `href` utility for path auto-completion and param validation. Alem Tuzlak

🛠  Code, Tools & Libraries

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/hih42ue0svxxvnopxjfu.jpg)](https://react.statuscode.com/link/166180/5d0bbb4a61)

[React Explorer: A Way to Visualize any React Powered Site](https://react.statuscode.com/link/166180/5d0bbb4a61 "www.react-explorer.com") — A new tool from React Scan’s creator for turning React sites into interactive visualizations representing the component tree. Highly experimental and new, so your mileage (and whether or not you even find such a visualization helpful) may vary.

Aiden Bai

🔎 If you like visualizations like the above, [Repofiles](https://react.statuscode.com/link/166181/5d0bbb4a61) offers a similar thing for GitHub repo structure. For example, [this visualization of the main React repo.](https://react.statuscode.com/link/166182/5d0bbb4a61)

[Redux Toolkit 2.6: Now with RTK Query Infinite Query Support](https://react.statuscode.com/link/166183/5d0bbb4a61 "github.com") — The popular way to integrate Redux into a modern React app has added a much requested feature: support for _infinite queries_ (the ability to keep fetching additional pages of data for a given endpoint).

Mark Erikson

[Justd: A Set of Accessible React Components Styled with Tailwind](https://react.statuscode.com/link/166184/5d0bbb4a61 "getjustd.com") — A self-described ‘chill set of React components’ built on top of Adobe’s popular (and highly accessible) React Aria Components.

Irsyad A. Panjaitan

*   🗓️ [React Big Calendar 1.18](https://react.statuscode.com/link/166185/5d0bbb4a61) – GCal/Outlook-like calendar component. Now with React 19 support.
    
*   [React Markdown 10.0](https://react.statuscode.com/link/166186/5d0bbb4a61) – Component to render markdown. ([Demo.](https://react.statuscode.com/link/166187/5d0bbb4a61))
    
*   [React Native Gesture Handler 2.24](https://react.statuscode.com/link/166188/5d0bbb4a61) – Exposes platform native touch APIs.
    
*   [Monaco Editor for React 4.7](https://react.statuscode.com/link/166189/5d0bbb4a61) – It's the same editor VS Code uses.
    
*   [React Icons 5.5](https://react.statuscode.com/link/166190/5d0bbb4a61) – Popular icon sets ready for use in React apps.
    
*   📅 [React Datepicker 8.1](https://react.statuscode.com/link/166191/5d0bbb4a61) – Simple date picker component.
    
*   🔊 [useSound 5.0](https://react.statuscode.com/link/166192/5d0bbb4a61) – React hook for playing sound effects.
    
*   [Reagent 1.3](https://react.statuscode.com/link/166193/5d0bbb4a61) – A simple ClojureScript interface to React.
    
*   [Exome 2.8](https://react.statuscode.com/link/166194/5d0bbb4a61) – State manager for deeply nested states.
    
*   [Mantine 7.17](https://react.statuscode.com/link/166195/5d0bbb4a61) – Full featured React component suite.
    
*   [MUI X 7.27](https://react.statuscode.com/link/166196/5d0bbb4a61) – Popular React component suite.
