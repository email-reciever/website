Kent C. Dodds

[Next.js SaaS Starter: A Next.js Template for Webapps](https://react.statuscode.com/link/159963/5d0bbb4a61 "github.com") — A starter template for building a SaaS-style webapp using Next.js with authentication, Stripe integration, and a dashboard for users. It uses Postgres and Drizzle for the database and UI elements based on [shadcn/ui](https://react.statuscode.com/link/159964/5d0bbb4a61) and Tailwind.
