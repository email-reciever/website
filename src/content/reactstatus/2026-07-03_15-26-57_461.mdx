[Meta Releases Its _Astryx_ Design System](https://react.statuscode.com/link/187432/5d0bbb4a61 "astryx.atmeta.com") — Built on React and [StyleX](https://react.statuscode.com/link/187433/5d0bbb4a61), and under development at Meta for over 8 years (powering _“over 13,000 apps”_, they claim), Astryx provides [more than 160 components](https://react.statuscode.com/link/187434/5d0bbb4a61), [theming](https://react.statuscode.com/link/187435/5d0bbb4a61), [templates](https://react.statuscode.com/link/187436/5d0bbb4a61), and tooling. There's also [a playground](https://react.statuscode.com/link/187437/5d0bbb4a61) to experiment with.

Meta

💡 In [How Astryx Works](https://react.statuscode.com/link/187438/5d0bbb4a61), the Meta team explains the technical story behind the project, why they built it, and the basics of its operation.

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/32e8caac.png)](https://react.statuscode.com/link/187431/5d0bbb4a61)

[Agentic Frontend Development](https://react.statuscode.com/link/187431/5d0bbb4a61 "master.dev") — Join this interactive workshop and see how AI can help with the messy middle of frontend work. Covers practical patterns for prompting, reviewing, debugging, and iterating with Codex — to help design, build, and refine modern interfaces.

Master.dev sponsor

**IN BRIEF:**

*   [Storybook has introduced first-class support for TanStack projects](https://react.statuscode.com/link/187439/5d0bbb4a61) in the React ecosystem, including Router, Start, and Query.
    
*   Vercel continues to tease Next.js 16.3 (now in preview) with a look at [Turbopack progress](https://react.statuscode.com/link/187440/5d0bbb4a61) and [AI-related improvements](https://react.statuscode.com/link/187441/5d0bbb4a61).
    
*   Vercel has also [introduced _Vercel Services_](https://react.statuscode.com/link/187442/5d0bbb4a61). It provides a way to run multiple frameworks in one project, e.g. a Next.js frontend and a backend that might well be built in Python, Go, or Rust.
    

[The Hidden Cost of Hydration Mismatches](https://react.statuscode.com/link/187443/5d0bbb4a61 "3perf.com") — How a single hydration mismatch, combined with web font swapping, can push a React page’s Largest Contentful Paint ([LCP](https://react.statuscode.com/link/187444/5d0bbb4a61)) from green into the red.

Ivan Akulov

📄 [How We Made React Native Animations Smooth on Low-End Android](https://react.statuscode.com/link/187445/5d0bbb4a61) – A deep dive into Skia, Reanimated, and Hermes performance on budget Android. Vishesh Raheja

📺 [How Tanner Linsley Built TanStack to Outlive Him](https://react.statuscode.com/link/187446/5d0bbb4a61) – A _very_ smoothly produced 15-minute interview with the creator of [TanStack](https://react.statuscode.com/link/187447/5d0bbb4a61). The Build Log

📄 [Next.js Already Traces Your Requests: How to Export Them with OpenTelemetry](https://react.statuscode.com/link/187448/5d0bbb4a61) Kyle Tryon (Sentry)

📄 [I Let React Compiler Handle Memoization: Here's What Actually Broke](https://react.statuscode.com/link/187449/5d0bbb4a61) Isaac Okoro (LogRocket)

🛠  Code, Tools & Libraries

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/mbrvbof3q1ujqnfapog4.jpg)](https://react.statuscode.com/link/187450/5d0bbb4a61)

[mapcn: Ready-to-Use Customizable Map Components](https://react.statuscode.com/link/187450/5d0bbb4a61 "www.mapcn.dev") — Built by a Cloudflare design engineer, mapcn is a suite of map components for React based on [MapLibre](https://react.statuscode.com/link/187451/5d0bbb4a61) and styled with Tailwind. If you use `shadcn/ui`, you’ll be very comfortable here as it follows the same approach.

Anmoldeep Singh

💬 [`shadcn/ui` Adds New Components for Chat Interfaces](https://react.statuscode.com/link/187452/5d0bbb4a61 "ui.shadcn.com") — `MessageScroller`, `Message`, `Bubble`, `Attachment`, and `Marker` components join the suite, providing an easy way to create a messaging experience in your app.

shadcn Team

[termcn: A Suite of Beautiful Terminal Components for React](https://react.statuscode.com/link/187453/5d0bbb4a61 "www.termcn.dev") — Built on [Ink](https://react.statuscode.com/link/187454/5d0bbb4a61) and [OpenTUI](https://react.statuscode.com/link/187455/5d0bbb4a61) and distributed via the shadcn CLI, this is a comprehensive component library for _terminal_ UIs.

Aniket Pawar

[React Icons 5.7: Easily Bring Popular Icons Into Your React Projects](https://react.statuscode.com/link/187456/5d0bbb4a61 "react-icons.github.io") — Use imports to bring in only the icons your project needs from sets like Font Awesome, Typicons, GitHub’s [Octicons](https://react.statuscode.com/link/187457/5d0bbb4a61), and more. v5.7 bumps all the icon sets up to their latest versions.

Kamijin Fanta

📰 Classifieds

Real-time UI, stale data? The bottleneck is the query, not the component. [TimescaleDB](https://react.statuscode.com/link/187458/5d0bbb4a61) fixes it.

* * *

⚛️ You compose React from components. Compose your AI backend the same way — agents as typed APIs, native TypeScript SDK, 100+ recipes. [→ Star on GitHub](https://react.statuscode.com/link/187459/5d0bbb4a61).

*   [Ant Design 6.5](https://react.statuscode.com/link/187460/5d0bbb4a61) – The enterprise-focused design language/React UI library gets a smaller bundle size, new icons, and [a DESIGN.md file](https://react.statuscode.com/link/187461/5d0bbb4a61) for agents to use.
    
*   [Downshift 9.4](https://react.statuscode.com/link/187462/5d0bbb4a61) – Primitives to build WAI-ARIA compliant React autocomplete, combobox and select components. [v9.4 fixes](https://react.statuscode.com/link/187463/5d0bbb4a61) React Compiler compatibility.
    
*   [React Router 8.1](https://react.statuscode.com/link/187464/5d0bbb4a61) – Adds a way for `create-react-router` to install an agent skill, and instrumentation data now [includes extra metadata](https://react.statuscode.com/link/187465/5d0bbb4a61).
    
*   [TinyBase 9.0](https://react.statuscode.com/link/187466/5d0bbb4a61) – Bug fix and stability release for the reactive data store and sync engine backend.
    
*   [Expo SDK 57](https://react.statuscode.com/link/187467/5d0bbb4a61) – Brings React Native 0.86 to Expo.
    
*   [MUI / Material UI 9.2](https://react.statuscode.com/link/187468/5d0bbb4a61) and [MUI-X 9.8](https://react.statuscode.com/link/187469/5d0bbb4a61)
    
*   [React Native VisionCamera 5.1](https://react.statuscode.com/link/187470/5d0bbb4a61)
