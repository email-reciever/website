**IN BRIEF:**

*   TanStack Query's maintainer has [unveiled an RFC](https://react.statuscode.com/link/169256/5d0bbb4a61) to collapse several overlapping methods into just two: `query()` and `infiniteQuery()` to make it more intuitive when fetching data imperatively.
    
*   🗣️ The folks over on the React sub-Reddit got to [discussing whether the future of React in 2025 is still bright.](https://react.statuscode.com/link/169257/5d0bbb4a61)
    
*   [eslint-plugin-react-you-might-not-need-an-effect](https://react.statuscode.com/link/169258/5d0bbb4a61) might be our favorite project name this month. It's an ESLint plugin to catch unnecessary use of `useEffect`.
    
*   🤖 Vercel has [introduced a one-click AI crawler/scraper filter](https://react.statuscode.com/link/169259/5d0bbb4a61) for Vercel hosted apps.
    

▶  [Building a Weather App with GitHub Copilot's Agent Mode](https://react.statuscode.com/link/169260/5d0bbb4a61 "www.youtube.com") — While tools like Cursor and Windsurf might be getting most of the press, VS Code and GitHub Copilot’s new ‘agent mode’ make a very powerful combination. Microsoft’s Rob Lourens and GitHub’s Cassidy Williams share a brief (4 minute) look at what’s possible.

[React Chrono 2.7: Dynamic, Interactive Timeline Component](https://react.statuscode.com/link/169266/5d0bbb4a61 "react-chrono.prabhumurthy.com") — Supports [horizontal](https://react.statuscode.com/link/169267/5d0bbb4a61), [vertical](https://react.statuscode.com/link/169268/5d0bbb4a61), and vertical alternating timelines, complete with keyboard accessibility, support for nested timelines, auto-playing timelines slideshow-style, and more. [Good docs](https://react.statuscode.com/link/169269/5d0bbb4a61) too. [v2.7](https://react.statuscode.com/link/169270/5d0bbb4a61) improves its search and animation features.

Prabhu Murthy et al.

[Bippy: A Toolkit for Hacking into React Internals](https://react.statuscode.com/link/169271/5d0bbb4a61 "www.bippy.dev") — Bippy pretends to be React’s devtools to get access to various internals, without any code modifications needed. It’s used as part of Aiden’s higher-level [React Scan](https://react.statuscode.com/link/169272/5d0bbb4a61) tool.

Aiden Bai

[Basecoat: `shadcn/ui`'s Magic, But Minus React](https://react.statuscode.com/link/169273/5d0bbb4a61 "basecoatui.com") — [shadcn/ui](https://react.statuscode.com/link/169274/5d0bbb4a61) is a popular suite of well designed React components, but if you’ve wanted to use them in _other_ contexts too, Basecoat is for you. Basecoat ports the components to a more vanilla form you can use elsewhere.

Ronan Berder

*   [React Query Builder 8.7](https://react.statuscode.com/link/169275/5d0bbb4a61) – A query builder component. Now with Prisma, Sequelize, and Drizzle query format support.
    
*   [React Native Tab View 4.1](https://react.statuscode.com/link/169276/5d0bbb4a61) – Cross-platform tab view component.
    
*   [BlockNote 0.30](https://react.statuscode.com/link/169277/5d0bbb4a61) – 'Notion-style' block-based editor.
    
*   [MUI X 8.3](https://react.statuscode.com/link/169278/5d0bbb4a61) – Popular React component suite.
    
*   [React Router 7.6](https://react.statuscode.com/link/169279/5d0bbb4a61)

📢  Elsewhere in JavaScript

A roundup of some other interesting stories in the broader JavaScript landscape, in case you've missed them:

*   [The V8 team introduces us to _explicit resource management_](https://react.statuscode.com/link/169282/5d0bbb4a61) where you can use `using` and `await using` to provide functions that are automatically called when resources go out of scope.
    
*   [Parcel v2.15.0](https://react.statuscode.com/link/169283/5d0bbb4a61) has been released with a new transformer and minifier for HTML and SVG that also supports converting SVGs to JSX for use in React components.
    
*   Brian Clark shares some [best practices for creating a modern npm package.](https://react.statuscode.com/link/169284/5d0bbb4a61)
    
*   [PptxGenJS](https://react.statuscode.com/link/169285/5d0bbb4a61) is a JavaScript library for creating PowerPoint presentations from code.
    
*   Matt Smith reminds us of [the power of the spread and rest syntax `...`](https://react.statuscode.com/link/169286/5d0bbb4a61)
    
*   [Piny](https://react.statuscode.com/link/169287/5d0bbb4a61) is a new, commercial, visual editor targeting Astro, React, Next.js and Tailwind use cases.
