[Introducing the _React Foundation_](https://react.statuscode.com/link/175321/5d0bbb4a61 "react.dev") — Over the years, concerns have arisen over who ‘owns/controls’ React, what might happen if that were to change, etc. It was announced at React Conf 2025 that control of React and React Native will be moved from Meta to an independent foundation, backed initially by several corporate members including Amazon, Expo, Meta, and Microsoft.

Webster, Carroll, Savona and Alpert

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/0c050628.png)](https://react.statuscode.com/link/175320/5d0bbb4a61)

[Go from Monolith to Monorepo](https://react.statuscode.com/link/175320/5d0bbb4a61 "frontendmasters.com") — Join Mike North for this course on architecting maintainable, fast and light codebases. You'll learn how to refactor a codebase into a TypeScript monorepo using tools like Nx and Lerna — covering dependencies, formatting, linting, performance and more.

Frontend Masters sponsor

[React 19.2 Released](https://react.statuscode.com/link/175322/5d0bbb4a61 "react.dev") — One of those major/minor releases React is famous for, introducing several new features like &lt;Activity />, a way to hide and restore the UI and internal state of its children, `useEffectEvent`, and improvements to Chrome DevTools' performance profiles to see more about React’s scheduling and the tree of components it’s currently working with.

The React Team

📺 At React Conf 2025, Mofei Zhang gave [▶️ a quick tour of React 19.2 and its new features.](https://react.statuscode.com/link/175323/5d0bbb4a61)

🛠  Code, Tools & Libraries

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/pd59joisiqwqnb4stfjp.jpg)](https://react.statuscode.com/link/175331/5d0bbb4a61)

[Coinbase Open Sources CDS (Coinbase Design System)](https://react.statuscode.com/link/175331/5d0bbb4a61 "cds.coinbase.com") — The team behind Coinbase has open sourced (Apache 2) its cross-platform component library for React and React Native. There are lots of neat effects in the components, such as this [‘rolling number’ component](https://react.statuscode.com/link/175332/5d0bbb4a61) and even simpler things like [a multi-step process stepper.](https://react.statuscode.com/link/175333/5d0bbb4a61) – [GitHub repo.](https://react.statuscode.com/link/175334/5d0bbb4a61)

Coinbase

[Adding Sound Effects to Emojis in Video API Reference App for React](https://react.statuscode.com/link/175335/5d0bbb4a61 "developer.vonage.com") — Level up your React video app with the Vonage Video API. Add emoji reactions with sound effects for fun, engaging calls!

Vonage sponsor

*   🗾 [react-map-gl 8.1](https://react.statuscode.com/link/175336/5d0bbb4a61) – React friendly API wrapper around MapboxGL JS. [See it in action.](https://react.statuscode.com/link/175337/5d0bbb4a61)
    
*   🔊 [React Native Audio API 0.9](https://react.statuscode.com/link/175338/5d0bbb4a61) – Audio engine based on Web Audio API.
    
*   [json-edit-react 1.29](https://react.statuscode.com/link/175339/5d0bbb4a61) – Component for editing/viewing JSON/object data ([Demo.](https://react.statuscode.com/link/175340/5d0bbb4a61))
    
*   [React Admin 5.12](https://react.statuscode.com/link/175341/5d0bbb4a61) – Framework for building B2B frontend interfaces.
    
*   [React Stripe.js 5.2](https://react.statuscode.com/link/175342/5d0bbb4a61) – Components for Stripe.js and Stripe Elements.
    
*   [Reactylon 3.2](https://react.statuscode.com/link/175343/5d0bbb4a61) – Babylon.js-powered XR framework for React.
