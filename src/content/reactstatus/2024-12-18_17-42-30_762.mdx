🗓️ Top React Developments & Items of 2024

React had a good 2024, though it felt quieter than some years in terms of big developments and updates. Every technology has these moments, especially when it's dominant in its space, but nonetheless, a variety of things did happen:

*   2024 was framed by the React team's [React Labs February update](https://react.statuscode.com/link/163675/5d0bbb4a61) where they confirmed that [_React Compiler_](https://react.statuscode.com/link/163676/5d0bbb4a61) was under active development and that React 19 was very much on the way.
    
*   Robin Wieruch started the year by [reflecting on React trends in 2024](https://react.statuscode.com/link/163677/5d0bbb4a61) and, with hindsight, pretty much hit the mark.
    

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/2eb7b19d.png)](https://react.statuscode.com/link/163674/5d0bbb4a61)

[Introducing Authentication Support for React Router](https://react.statuscode.com/link/163674/5d0bbb4a61 "go.clerk.com") — Clerk announces beta support for React Router, including pre-built UI components, server utilities, and low level utilities for any of your custom flows. You can use React Router both as a framework or library with Clerk.

Clerk sponsor

*   [React Conf 2024](https://react.statuscode.com/link/163678/5d0bbb4a61) took place in the outskirts of Las Vegas in May. [Here's a recap](https://react.statuscode.com/link/163679/5d0bbb4a61) of the event.
    
*   [React Native's _New Architecture_](https://react.statuscode.com/link/163680/5d0bbb4a61) went live in React Native 0.76.
    
*   Remix and React Router [underwent a merger](https://react.statuscode.com/link/163681/5d0bbb4a61) _of sorts_ and [React Router v7](https://react.statuscode.com/link/163682/5d0bbb4a61) landed in November.
    
*   [Next.js 15 was released](https://react.statuscode.com/link/163683/5d0bbb4a61) with React 19 and React Compiler support (and even more so in [Next.js 15.1](https://react.statuscode.com/link/163684/5d0bbb4a61)).
    
*   We got some new React performance related tools in the shape of [React Geiger](https://react.statuscode.com/link/163685/5d0bbb4a61), [React Scan](https://react.statuscode.com/link/163686/5d0bbb4a61), and [Million Lint.](https://react.statuscode.com/link/163687/5d0bbb4a61)
    
*   🕹️ A full-scale React powered game, [Athena Crisis](https://react.statuscode.com/link/163688/5d0bbb4a61), available [on Steam](https://react.statuscode.com/link/163689/5d0bbb4a61), went open source.
    
*   [Material UI v6](https://react.statuscode.com/link/163690/5d0bbb4a61) arrived just in time to celebrate its 10th birthday.
    
*   [The final release of React v19](https://react.statuscode.com/link/163691/5d0bbb4a61) rounded out the year, landing just over a week ago now.
    

🥇 Top Articles of the Year

Now for a quick look at the top six links we included in the newsletter (skipping things already mentioned above), based on reader engagement:

1. [How to Start a React Project in 2024](https://react.statuscode.com/link/163692/5d0bbb4a61 "www.robinwieruch.de") — It seems appropriate we started out 2024 with a look at the pros and cons of a few approaches to kicking off a React app. I bet we'll be getting a 2025 edition soon too!

Robin Wieruch

2. [React 19 Cheat Sheet](https://react.statuscode.com/link/163693/5d0bbb4a61 "www.epicreact.dev") — From Kent C Dodds of [Epic React](https://react.statuscode.com/link/163694/5d0bbb4a61) fame came this part cheat sheet and part reminder of some of the things you can do in React 19.

Kent C. Dodds

3. [Why Use Redux Today?](https://react.statuscode.com/link/163695/5d0bbb4a61 "blog.isquaredsoftware.com") — The maintainer of the popular [Redux](https://react.statuscode.com/link/163696/5d0bbb4a61) state management library gave a talk at _React Summit 2024_ where he gave a thorough introduction to the history of Redux, how it grew, and the benefits it continues to provide. [There are detailed slides](https://react.statuscode.com/link/163697/5d0bbb4a61) if you’d prefer not to watch the video.

Mark Erikson

4. [Enhancing _The New York Times_' Web Performance with React 18](https://react.statuscode.com/link/163698/5d0bbb4a61 "open.nytimes.com") — Late in 2023, the team at _The New York Times_ set out to take full advantage of React 18 on its flagship news site. This was an interesting case study looking at the challenges faced in upgrading, and the significant benefits they managed to take advantage of.

Ilya Gurevich (NYT)

5. [Component Composition is Great, BTW](https://react.statuscode.com/link/163699/5d0bbb4a61 "tkdodo.eu") — A well-regarded author writes about using component composition to avoid conditional rendering for mutually exclusive states (and why we should all be doing the same).

Dominik Dorfmeister (AKA TkDodo)

6. [Bulletproof React: A Scalable Architecture for Production Apps](https://react.statuscode.com/link/163700/5d0bbb4a61 "github.com") — A popular opinionated guide to how you can structure a large scale React app. It’s had a few major updates over the years, but notably now includes a Next.js version, with both app router and page router approaches.

Alan Alickovic

⚙️ Top Repos of the Year

1. [Next.js SaaS Starter: A Next.js Template for Webapps](https://react.statuscode.com/link/163701/5d0bbb4a61 "github.com") — A starter template for building a SaaS-style webapp using Next.js with support for authentication, Stripe integration, and a dashboard for logged-in users. It uses Postgres and Drizzle for the database and UI elements based on shadcn/ui and Tailwind.

Lee Robinson (Vercel)

2. [Functional UI Kit: A Unified Figma and React UI Component Kit](https://react.statuscode.com/link/163702/5d0bbb4a61 "functional-ui-kit.com") — Despite many attempts, it doesn’t feel like anyone has really perfected the Figma - React connection yet, but this was a good attempt to bridge the design/code gap, funded by Figma’s [Community Creators Fund.](https://react.statuscode.com/link/163703/5d0bbb4a61) No updates in several months, however..

Alex Yakir et al.

3. [MistCSS: Build Components with Just CSS](https://react.statuscode.com/link/163704/5d0bbb4a61 "typicode.github.io") — Forget about CSS-in-JS, what about JS-_from_\-CSS? This approach provided a way to compile CSS-only React components into a TypeScript equivalent, but as of the recent v1.0 release it went a step further and doesn't generate or require any JavaScript at all.. A neat idea.

Typicode

4. [assistant-ui: React Components for AI Chat](https://react.statuscode.com/link/163705/5d0bbb4a61 "github.com") — It doesn’t just give you the interface components, but has integrations for Vercel AI, Langchain, and talking to other common LLM APIs (e.g. OpenAI’s), enabling you to build your own in-house AI chat system quickly. [Examples.](https://react.statuscode.com/link/163706/5d0bbb4a61)

Simon Farshid

5. [Puck: A Self-Hosted Drag-and-Drop Editor for React](https://react.statuscode.com/link/163707/5d0bbb4a61 "github.com") — Sitting somewhere between an old-school WYSIWYG-powered CMS and a headless one, Puck enables content teams to author content using real React components. Want to see a demo? [Of course](https://react.statuscode.com/link/163708/5d0bbb4a61).

🗓️ 2024 年 React 顶级开发和项目

React 的 2024 年表现不错，尽管在重大开发和更新方面感觉比前几年安静一些。每项技术都有这些时刻，尤其是当它在其领域占据主导地位时，但尽管如此，还是发生了各种各样的事情：

* 2024 年是由 React 团队的 [React Labs 2 月更新](https://react.statuscode.com/link/163675/5d0bbb4a61) 确定的，他们确认 \[*React Compiler*]\(https://react.statuscode. com/link/163676/5d0bbb4a61）正在积极开发中，React 19 即将推出。
