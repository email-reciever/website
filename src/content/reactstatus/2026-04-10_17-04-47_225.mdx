[Ink 7.0: Use React to Build TUIs and Command Line Apps](https://react.statuscode.com/link/183559/5d0bbb4a61 "github.com") — A great example of _‘right place, right time’_, Ink now powers [tons of apps](https://react.statuscode.com/link/183560/5d0bbb4a61), especially in the agentic AI ecosystem. v7.0 now leans on React 19.2, uses `useEffectEvent` internally for added efficiency, and brings many new hooks and settings.

Tanner Linsley

[Under the Hood of MDN's New Frontend](https://react.statuscode.com/link/183573/5d0bbb4a61 "developer.mozilla.org") — [MDN](https://react.statuscode.com/link/183574/5d0bbb4a61) remains one of the most useful resources for frontend developers and it has a new frontend. This case study digs deep into how they replaced an ejected _Create React App_ based app with one driven by [Lit](https://react.statuscode.com/link/183575/5d0bbb4a61)\-based web components.

Leo McArdle (MDN)
