📢  Elsewhere in JavaScript

A quick roundup of some of other interesting stories in the broader JavaScript landscape, in case you've missed them:

*   [TypeScript 5.8 Beta](https://react.statuscode.com/link/165318/5d0bbb4a61) has been released – it's mostly exciting for Node.js users, but there's a new [`--erasableSyntaxOnly` option](https://react.statuscode.com/link/165319/5d0bbb4a61) to only allow the use of TypeScript features that can be safely erased from the source (e.g. not enums).
    
*   🤣 [The Deno vs Oracle JavaScript™ trademark case rolls on](https://react.statuscode.com/link/165320/5d0bbb4a61) rolls on with Oracle just barely meeting a deadline to [file a motion to dismiss the case.](https://react.statuscode.com/link/165321/5d0bbb4a61) Oracle asserts that _"relevant consumers do not perceive JAVASCRIPT as a generic term."_ We don't? Or aren't we relevant?
    
*   [Centrifugo v6 has been released.](https://react.statuscode.com/link/165322/5d0bbb4a61) Centrifugo is a Go-powered but otherwise language agnostic real-time messaging server for wiring up asynchronous, multi-user apps.
    
*   Josh Goldberg explains [the differences between ESLint and TypeScript,](https://react.statuscode.com/link/165323/5d0bbb4a61) beyond the obvious.
    
*   [Bun 1.2.2](https://react.statuscode.com/link/165324/5d0bbb4a61) has been released with big drops in idle memory usage.
