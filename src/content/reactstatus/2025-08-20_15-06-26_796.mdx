Zilberman, Zaidman, et al.

🛠  Code, Tools & Libraries

Matt Abrams (Builder)

🤖 [React ChatBotify: React Library for Building Chatbot Systems](https://react.statuscode.com/link/173048/5d0bbb4a61 "react-chatbotify.com") — A platform and library for building modern chatbot tools for the Web. Auth0 sponsor

[React Cosmos 7.0: Sandbox for Developing & Testing UI Components](https://react.statuscode.com/link/173051/5d0bbb4a61 "reactcosmos.org") — A React-only tool for developing and testing UI components in isolation. It’s [explained in more depth here.](https://react.statuscode.com/link/173052/5d0bbb4a61)

Ovidiu Cherecheș et al.

[React Native WebGPU](https://react.statuscode.com/link/173053/5d0bbb4a61 "github.com") — A React Native WebGPU implementation using [Dawn.](https://react.statuscode.com/link/173054/5d0bbb4a61) Currently a technical preview.

William Candillon

*   [Waku 0.25](https://react.statuscode.com/link/173055/5d0bbb4a61) – The minimal React framework introduces the concept of 'slice components' – a new approach to fine-grained component rendering.
    
*   ✋ [React Native Gesture Handler 2.28](https://react.statuscode.com/link/173056/5d0bbb4a61) – Exposes platform native touch APIs. Now with [React Native 0.81](https://react.statuscode.com/link/173041/5d0bbb4a61) support.
    
*   🗓️ [React Date Picker 8.6](https://react.statuscode.com/link/173057/5d0bbb4a61) – Simple date picker component. ([Demo](https://react.statuscode.com/link/173058/5d0bbb4a61).)
    
*   ♟️ [React Chessboard 5.5](https://react.statuscode.com/link/173059/5d0bbb4a61) – Render chess boards. ([Examples.](https://react.statuscode.com/link/173060/5d0bbb4a61))
    
*   [React Slick 0.31](https://react.statuscode.com/link/173061/5d0bbb4a61) – A React component port of [Slick Carousel.](https://react.statuscode.com/link/173062/5d0bbb4a61)
    
*   [Enact 5.2](https://react.statuscode.com/link/173063/5d0bbb4a61) – App development framework built atop React.
    

📢  Elsewhere in JavaScript

A roundup of some other interesting stories in the broader JavaScript landscape, in case you've missed them:

*   [Porffor](https://react.statuscode.com/link/173064/5d0bbb4a61) is a novel ahead-of-time compiler for JavaScript that can compile both to WebAssembly and native binaries. It can now also be used to [run JavaScript apps on AWS Lambda without long 'cold start' times.](https://react.statuscode.com/link/173065/5d0bbb4a61)
    
*   [TypeGPU](https://react.statuscode.com/link/173066/5d0bbb4a61) is a new TypeScript library that enhances the WebGPU API, allowing resource management in a type-safe, declarative way. It can be used with React Native too, via [react-native-webgpu](https://react.statuscode.com/link/173053/5d0bbb4a61).
    
*   📺 TC39's Daniel Ehrenberg took part in [▶️ an informative 47-minute interview](https://react.statuscode.com/link/173067/5d0bbb4a61) where he spoke about how the TC39 committee works and how _you_ can have your say on the future of JavaScript.
    
*   [Oxlint now has type-aware linting in preview.](https://react.statuscode.com/link/173068/5d0bbb4a61)
    
*   [pnpm 10.15](https://react.statuscode.com/link/173069/5d0bbb4a61) has been released.
