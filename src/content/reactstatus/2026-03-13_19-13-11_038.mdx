[Vite 8.0 Released](https://react.statuscode.com/link/182180/5d0bbb4a61 "vite.dev") — A mega release for the popular build tool. Designed to be a smooth upgrade, there’s a _lot_ going on behind the scenes: `@vitejs/plugin-react` v6 no longer needs Babel (Oxc steps up), [Rolldown](https://react.statuscode.com/link/182181/5d0bbb4a61) replaces Rollup and esbuild, Wasm SSR support, browser console forwarding to the terminal, and huge perf improvements. Great news for Remix, TanStack Start, and Astro users!

Vite

💡 [Vitest 4.1](https://react.statuscode.com/link/182183/5d0bbb4a61) has also been released with Vite 8 support.

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/cf4d5cbe.png)](https://react.statuscode.com/link/182179/5d0bbb4a61)

[Expo Router v55 Makes React Devs at Home in React Native](https://react.statuscode.com/link/182179/5d0bbb4a61 "expo.dev") — Expo Router ships iOS, Android, and web from one file-based routing tree. No WebViews, no pkg-spaghetti, just sensible defaults and carefully designed components.

Expo sponsor

[How we Rewrote 130K Lines from React to Svelte in Two Weeks](https://react.statuscode.com/link/182185/5d0bbb4a61 "strawberrybrowser.com") — [Over on Bluesky](https://react.statuscode.com/link/182189/5d0bbb4a61), Rich Harris referenced the idea that LLMs could lock everyone into the most popular framework, but in reality they make _switching between frameworks_ easier than ever before.

**IN BRIEF:**

*   [Dan Abramov noted](https://react.statuscode.com/link/182193/5d0bbb4a61) that pages on the React site have a _'Copy Page'_ button that copies the page as Markdown (which you could feed to an agent, say). You can also add `.md` to the URL, [like in this example.](https://react.statuscode.com/link/182197/5d0bbb4a61)
    
*   In [this _X_ post](https://react.statuscode.com/link/182201/5d0bbb4a61), Joe Savona of the React team says that a Rust-powered version of the React Compiler is coming soon.
    
*   Interactive RSC sandboxes are [coming to the official React docs.](https://react.statuscode.com/link/182204/5d0bbb4a61)
    
*   If you use [React Admin](https://react.statuscode.com/link/182208/5d0bbb4a61), enjoy [this roundup of recent developments.](https://react.statuscode.com/link/182212/5d0bbb4a61)
    
*   [TypeScript 6.0 Release Candidate](https://react.statuscode.com/link/182216/5d0bbb4a61) has been released.
    

[shadcn/cli v4 Released](https://react.statuscode.com/link/182220/5d0bbb4a61 "ui.shadcn.com") — [shadcn/ui](https://react.statuscode.com/link/182223/5d0bbb4a61)'s [CLI tool](https://react.statuscode.com/link/182277/5d0bbb4a61) gets an upgrade aimed at humans _and_ agents with 'skills' to improve agent performance, _presets_ to pack a design system config into a simple ID (and switch between them), 'dry runs' to see what a registry will add prior to doing it, and `shadcn/create` is rebuilt with a set of components for fast preset previews.

shadcn

[An Empirical Study of Frontend Memory Leaks](https://react.statuscode.com/link/182230/5d0bbb4a61 "stackinsight.dev") — Analysis of 500 repos (210 of which were React apps) for common patterns that lead to memory leaks. Missing timer cleanups and event listener removals made up the majority of problems found.

Ko-Hsin Liang

[18 Months of Code, Gone: Here's What We Learned](https://react.statuscode.com/link/182233/5d0bbb4a61 "tompiagg.io") — Why a development team decided to discard Next.js and Server Actions, opting for a new stack oriented around TanStack Start and Hono.

🛠  Code, Tools & Libraries

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/ggrchqegivs99f4ohysg.jpg)](https://react.statuscode.com/link/182248/5d0bbb4a61)

[RedwoodSDK 1.0 Released: The Cloudflare-Native React Framework](https://react.statuscode.com/link/182248/5d0bbb4a61 "rwsdk.com") — RedwoodSDK, essentially a full reboot of the former _RedwoodJS_, is a server-first React framework, built as a Vite plugin, that integrates deeply with the Cloudflare platform ([why?](https://react.statuscode.com/link/182249/5d0bbb4a61)) and its provision of workers, databases (D1), durable objects, storage (R2), AI APIs, etc.

Peter Pistorius

[Reveal.js 6.0: The HTML Presentation Framework Adds React](https://react.statuscode.com/link/182251/5d0bbb4a61 "revealjs.com") — A long-standing way to bring elegant presentations to anyone with a browser. [v6.0](https://react.statuscode.com/link/182252/5d0bbb4a61) notably introduces an official React wrapper [explained in detail here](https://react.statuscode.com/link/182253/5d0bbb4a61), meaning you can build your slides in React component form.

Hakim El Hattab

[Bippy: A Toolkit for Hacking into React Internals](https://react.statuscode.com/link/182250/5d0bbb4a61 "www.bippy.dev") — Pretends to be React’s devtools to get access to various internals, without needing any code modifications on your app’s part.

Aiden Bai

[React Native View Recorder: Capture Views to a Video or Image](https://react.statuscode.com/link/182254/5d0bbb4a61 "react-native-view-recorder.awingender.com") — Capture any React Native View to a video or an image. Add your own audio. Ideal for creating shareable content in your apps. [GitHub repo.](https://react.statuscode.com/link/182255/5d0bbb4a61)

Antonín Wingender

[Number Flow React Native: Beautiful Number Animations](https://react.statuscode.com/link/182256/5d0bbb4a61 "number-flow-react-native.awingender.com") — High-quality number animations for React Native apps, with digit-by-digit rolling counters and `Intl.NumberFormat` support. [GitHub repo.](https://react.statuscode.com/link/182257/5d0bbb4a61)

Antonín Wingender

[Hit Area: Tailwind CSS Utility Classes to Expand Hit Areas](https://react.statuscode.com/link/182258/5d0bbb4a61 "bazza.dev") — Adds `hit-area-*` utility classes to increase the area around controls which response to user actions. Distributed via the shadcn registry.

Kian Bazarjani

*   [React Aria 1.16.0](https://react.statuscode.com/link/182259/5d0bbb4a61) – Adobe's suite of accessible components adds multi-select support to its combobox. [Try it here](https://react.statuscode.com/link/182260/5d0bbb4a61) _(select 'multiple' on the right)._
    
*   [Astro 6.0](https://react.statuscode.com/link/182261/5d0bbb4a61) – A big update for the popular content site framework, including a built-in Fonts API, Content Security Policy API, a redesigned `astro dev` using Vite's new Environment API, and more.
    
*   [React Modal Sheet 5.4](https://react.statuscode.com/link/182262/5d0bbb4a61) – Flexible bottom sheet component built with Motion.
    
*   [react-native-nitro-sqlite 9.6](https://react.statuscode.com/link/182263/5d0bbb4a61) – Fast SQLite library for React Native.
    
*   [react-share v5.3.0](https://react.statuscode.com/link/182264/5d0bbb4a61) – Social media share buttons for React apps.
    
*   📊 [Recharts 3.8](https://react.statuscode.com/link/182265/5d0bbb4a61) – D3-powered React charting library.
    
*   [Preact 10.29.0](https://react.statuscode.com/link/182266/5d0bbb4a61) – The fast 3KB React alternative.
    
*   [BaseUI 1.3](https://react.statuscode.com/link/182267/5d0bbb4a61) – Unstyled UI component library.

*   🕒 **It's about time…** At [this week's TC39 meeting](https://react.statuscode.com/link/182271/5d0bbb4a61), the Temporal proposal progressed to Stage 4.
