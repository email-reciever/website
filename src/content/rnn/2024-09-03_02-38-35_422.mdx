# This issue: Chain React 2024 videos released, Shopify just open-sourced Tophat, the power of Nitro Modules, and why EAS is about to get cheaper.

  
Welcome to the August issue of the React Native Newsletter!  
  
I've got some great news for you. All videos from [Chain React 2024 in Portland, OR](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=ecebd48529&e=077ac9fa1d), have been released and are available [here](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=40199db434&e=077ac9fa1d).  
  
Have you heard of [Tophat](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=5596d72d0c&e=077ac9fa1d)? It's [Shopifiy](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=35f8f1005c&e=077ac9fa1d)'s solution for installing and testing any mobile application, and they've released it as open source!  
  
Be sure to check out this [game-changer X post](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=1428e4b1b6&e=077ac9fa1d) from [@mrousavy](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=51ca952b7d&e=077ac9fa1d) about using Nitro Modules for writing first-class Swift support, insanely fast C++ bindings, object-oriented/instance-based approach, and much more.  
  
Coming up early September, 2024: [Callstack's](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=d179c7d28f&e=077ac9fa1d) [React Universe Conference 2024](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=0f5188b0ed&e=077ac9fa1d) in Wrocław, Poland, held September 5-6. This year, [Infinite Red](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=815cd0ea1d&e=077ac9fa1d)'s co-host of the React Native Radio podcasts, [Mazen Chami](https://red.us3.list-manage.com/track/click?u=78d9e37a94fa0b522939163d4&id=8427adcd31&e=077ac9fa1d), will deliver a talk. Take a look at the Top React Native picks section below to find a 15% discount code for purchasing tickets to this exciting event!
