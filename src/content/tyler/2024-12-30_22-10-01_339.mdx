## Pop Quiz: Answer

#### Sponsored by [StackBlitz](https://click.convertkit-mail4.com/p9unqxnv5mt9h2km3w4urupz86933hr/25h2h9u2qk7d5nt3/aHR0cHM6Ly94LmNvbS9zdGFja2JsaXR6L3N0YXR1cy8xODY5NzE1NjYxNDQ0MDQzMjQ1)

You can use an `AbortController` to remove all the event listeners at once. When passed a `signal` as an option, the event listener will be removed when the `AbortSignal`’s `abort()` method is called.

```js
const controller = new AbortController();
const &lcub; signal } = controller;

element.addEventListener('click', () => &lcub;
  console.log('clicked');
}, &lcub; signal });

element.addEventListener('mouseover', () => &lcub;
  console.log('hovered');
}, &lcub; signal });

element.addEventListener('mouseout', () => &lcub;
  console.log('unhovered');
}, &lcub; signal });

element.addEventListener('focus', () => &lcub;
  console.log('focused');
}, &lcub; signal });

controller.abort();
```

Note: Older browsers may not support the `options` parameter for `addEventListener`, but it is supported in all modern browsers.
