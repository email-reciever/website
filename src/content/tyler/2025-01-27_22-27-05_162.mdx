What color is the `&lt;p>` tag’s text?

```html
&lt;p class="hello">Hello&lt;/p>

&lt;style>
  body &lcub; color: red; }
  p &lcub; color: green; }
  .hello &lcub; color: unset; }
&lt;/style>
```

The answer is `red` since `unset` “unsets” the style of the `p` tag, so it inherits from its parent – `body`.
