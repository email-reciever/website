## Cool Bits

1.  Armin Ronacher wrote about [why we should call agents “clankers”](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/e0hph0u7gqv456h8/aHR0cHM6Ly9sdWN1bXIucG9jb28ub3JnLzIwMjYvNS8yNi9jbGFua2Vycy8=). We all know he’s not invited to the cookout after the robot uprising.
    
2.  The React Aria team just released version 1.18.0, which has new features for `Calendar` like multiple date selection, updates to `Slider` that make it easier to style, and some sourdough yeast Devon had leftover after COVID.
    
3.  Vandana Verma Sehgal from Snyk is doing a [live workshop June 4 on MCP security](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/7qh7h2u97wlz3lsz/aHR0cHM6Ly9nby5zbnlrLmlvL293YXNwLW1jcC10b3AtMTAtaXNjMi5odG1sP3V0bV9jYW1wYWlnbj1kbV9wcF9ieXRlcy1hZHZlcnRpc2VyXzI2MDUyOV93Ym5fMjYwNjA0X293YXNwLW1jcC10b3AtMTAmdXRtX21lZGl1bT1lbS1wYSZ1dG1fc291cmNlPWJ5dGVzLWFkdmVydGlzZXImdXRtX2NvbnRlbnQ9d2JuXzI2MDYwNF9vd2FzcC1tY3AtdG9wLTEw). You’ll learn how to avoid MCP failure modes like prompt injection, excessive permissions, unsafe tool execution, and context leakage. \[sponsored\]
    
4.  Nolan Lawson makes the case that LLMs don’t have to be slop cannons and that you can use them [to write better code more slowly](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/owhkhwuwvg9o5dsv/aHR0cHM6Ly9ub2xhbmxhd3Nvbi5jb20vMjAyNi8wNS8yNS91c2luZy1haS10by13cml0ZS1iZXR0ZXItY29kZS1tb3JlLXNsb3dseS8=).
    
5.  If you combine Elm’s architecture, Effect’s type safety guarantees, and TypeScript, you get the most pretentious front end framework of all time [foldkit.dev](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/z2hgh7uevp6w0vfp/aHR0cHM6Ly9mb2xka2l0LmRldi8=), “the front end framework for correctness”.
    
6.  Callstack made [a custom coding model for React Native](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/p8hehqu40g6x8waq/aHR0cHM6Ly93d3cuY2FsbHN0YWNrLmNvbS9ibG9nL2ludHJvZHVjaW5nLWFwZXgtYS1mYXN0LXNwZWNpYWxpemVkLW1vZGVsLWZvci1yZWFjdC1uYXRpdmU=). It knows framework conventions and cross platform details, but stops working if you call it a cl\*nker.
    
7.  Clerk added [Application Logs to their dashboard](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/6qhehoulgnro09bo/aHR0cHM6Ly9nby5jbGVyay5jb20vM1VtUUtyeA==) so you can track things like sign-ins, sign-ups, user updates, organization changes, billing events, and more. \[sponsored\]
    
8.  Andrey Sitnik (the author of `postcss` and `autoprefixer`) wrote a helpful article on [how to stop burning tokens teaching your coding agent how to use pre-commit hooks](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/kkhmh2unq6wm4osl/aHR0cHM6Ly9ldmlsbWFydGlhbnMuY29tL2Nocm9uaWNsZXMvc3RvcC13cml0aW5nLXJ1bGVzLWluLWFnZW50cy1tZC11c2UtYWdlbnQtaG9va3MtYW5kLW5hbm8tc3RhZ2VkLWluc3RlYWQ=).
    
9.  Sunkanmi Fafowora dove into [the age-old problem of styling checkmarks](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/58hvh8ug8kx7lwi6/aHR0cHM6Ly9waWNjYWxpbC5saS9ibG9nL25hdmlnYXRpbmctdGhlLWFnZS1vbGQtcHJvYmxlbS1vZi1jaGVja21hcmtzLWluLXVpLXdpdGgtcHJvZ3Jlc3NpdmUtZW5oYW5jZW1lbnQv).
    
10.  Convex made a video on [data access patterns for modern web applications](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/25h2h9u3z49dn5c3/aHR0cHM6Ly93d3cueW91dHViZS5jb20vd2F0Y2g_dj16VGxrWTVVRG1RTSZ0PTJz) and the architectural tradeoffs around row-level security. \[sponsored\]
    
11.  Astro jumped on the “rewrite it in Rust” hype train and [overhauled their markdown processor](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/qvh8h8ud5ownegul/aHR0cHM6Ly9hc3Ryby5idWlsZC9ibG9nL2FzdHJvLTY0MC8=) in version 6.4.
    
12.  Nanda Syahrasyad wrote a step-by-step guide on [how to build a key-value database from scratch](https://c5e21242.click.convertkit-mail.com/e5um2zm0q3b7hl3er9pa8h85q5gg8tlhzrqn/g3hnhwum569pgdcr/aHR0cHM6Ly93d3cubmFuLmZ5aS9kYXRhYmFzZQ==). Read at your own risk because when front end devs try and database, it does not end well.
