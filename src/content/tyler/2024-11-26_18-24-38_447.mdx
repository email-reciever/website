## Pop Quiz: Answer

You can use the element’s `getAnimations` method to get an array of all the animations on the element. From there you can loop through the array and call the `pause()` method on each animation.

```js
const circle = document.querySelector(".circle");
const animations = circle.getAnimations();

animations.forEach(animation => &lcub;
  animation.pause();
})
```

Here’s what the full solution would look like, along with a [working example](https://click.convertkit-mail4.com/r8u7647ndptoh28k75xc6ud708666h7/dphehmu0p83n7zfm/aHR0cHM6Ly9jb2Rlc2FuZGJveC5pby9zL2J5dGVzLXBhdXNlLWFuaW1hdGlvbi1pNXUyZXk_ZmlsZT0lMkZpbmRleC5odG1s) if you’re the curious type.

```html
&lt;style>
  .circle &lcub;
    width: 100px;
    height: 100px;
    border-radius: 50%;
    background-color: lightblue;
    margin: auto;
    animation: pulse 1s infinite ease-in-out;
  }

  @keyframes pulse &lcub;
    0% &lcub;
      transform: scale(1);
    }
    50% &lcub;
      transform: scale(1.2);
    }
    100% &lcub;
      transform: scale(1);
    }
  }
&lt;/style>

&lt;div class="circle">&lt;/div>
&lt;button class="btn">Pause Animation&lt;/button>

&lt;script>
  const btn = document.querySelector(".btn");
  
  btn.addEventListener("click", () => &lcub;
    const animations = document.querySelector(".circle")
      .getAnimations();
    
    if (btn.textContent === "Pause Animation") &lcub;
      animations.forEach((animation) => &lcub;
        animation.pause();
      });
      btn.textContent = "Play Animation";
    } else &lcub;
      animations.forEach((animation) => &lcub;
        animation.play();
      });
      btn.textContent = "Pause Animation";
    }
  });
&lt;/script>
```

There are other useful properties you can access on the animation object such as `currentTime` and `playbackRate` to have even more control over the animation.

## 小测验：答案

您可以使用元素的“getAnimations”方法来获取元素上所有动画的数组。从那里您可以循环遍历数组并在每个动画上调用“pause()”方法。

```js
const circle = document.querySelector(".circle");
const animations = circle.getAnimations();

animations.forEach(animation => &lcub;
  animation.pause();
})
```

这是完整的解决方案，以及\[工作示例]\(https://click.convertkit-mail4.com/r8u7647ndptoh28k75xc6ud708666h7/dphehmu0p83n7zfm/aH R0chHM6Ly9jb2Rlc2FuZGJveC5pby9zL2J5dGVzLXBhdXNlLWFuaW1hdGlvbi1pNXUyZXk\_ZmlsZT0lMkZpbmRleC5odG1s)如果你是好奇的类型。

```html

  .circle &lcub;
    width: 100px;
    height: 100px;
    border-radius: 50%;

    margin: auto;

  }

  @keyframes pulse &lcub;
    0% &lcub;
      transform: scale(1);
    }
    50% &lcub;
      transform: scale(1.2);
    }
    100% &lcub;
      transform: scale(1);
    }
  }
&lt;/style>

&lt;div class="circle">&lt;/div>
&lt;button class="btn">Pause Animation&lt;/button>

  btn.addEventListener("click", () => &lcub;
    const animations = document.querySelector(".circle")
      .getAnimations();
    
    if (btn.textContent === "Pause Animation") &lcub;
      animations.forEach((animation) => &lcub;
        animation.pause();
      });
      btn.textContent = "Play Animation";
    } else &lcub;
      animations.forEach((animation) => &lcub;
        animation.play();
      });
      btn.textContent = "Pause Animation";
    }
  });
&lt;/script>
```

您可以在动画对象上访问其他有用的属性，例如“currentTime”和“playbackRate”，以更好地控制动画。
