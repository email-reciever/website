## Cool Bits

1.  shadcn just launched [Charts](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/m2h7h6u85mx3xkhm/aHR0cHM6Ly91aS5zaGFkY24uY29tL2NoYXJ0cw==) – a collection of beautiful, copy-pasteable chart components that are built on top of Recharts. If you thought all SAAS apps already looked the same, buckle up.
    
2.  Eric Allam wrote a deep dive about how he and his team [tamed Node.js event loop lag](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/dphehmu068xex4tm/aHR0cHM6Ly90cmlnZ2VyLmRldi9ibG9nL2V2ZW50LWxvb3AtbGFn).
    
3.  React Rally is one of the most beloved React confs ever, and they’re offering [a 25% discount to all Bytes subscribers through this link](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/e0hph0u0vqz7zlc8/aHR0cHM6Ly90aS50by9yZWFjdC1yYWxseS9yZWFjdC1yYWxseS0yMDI0L2Rpc2NvdW50L0J5dGVz). Come to Park City, Utah on Aug 12-14th and learn from Chris Sev, Kent C. Dodds, Shruti Kapoor, and many more. \[sponsored\]
    
4.  [es-toolkit](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/owhkhwur9geweluv/aHR0cHM6Ly9naXRodWIuY29tL3Rvc3MvZXMtdG9vbGtpdA==) is a new JavaScript utility library that bills itself as a faster, more modernized version of lodash, which is a faster, more modernized version of underscore.
    
5.  [matcha.css](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/z2hgh7u36pzez8ip/aHR0cHM6Ly9naXRodWIuY29tL2xvd2xpZ2h0ZXIvbWF0Y2hh) is a pure CSS library designed to style HTML elements similar to a default browser stylesheet. But as much as I _want_ to like matcha, it always ends up tasting like grass juice to me.
    
6.  [React, Visualized](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/p8hehquz6ge4e7sq/aHR0cHM6Ly9yZWFjdC5nZy92aXN1YWxpemVkP3M9Ynl0ZXM=) is our gentle introduction to React, but visualized.
    
7.  Sentry’s [JavaScript error and performance monitoring tools](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/x0hph3uw29memnf5/aHR0cHM6Ly9zZW50cnkuaW8vZm9yL2phdmFzY3JpcHQv) give you max power with minimum effort (my favorite). You get React 19 support, auto-config for dozens of libraries, and a one-line setup that makes it super easy to get started (for free). \[sponsored\]
    
8.  [Flitter](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/6qhehou7rn4l47bo/aHR0cHM6Ly9mbGl0dGVyLnBhZ2VzLmRldi8=) is a JavaScript data visualization library that uses Flutter-like syntax – aka the crossover that no one asked for. But it does look a lot more promising than this [crossover flop](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/kkhmh2ulw6pnpnhl/aHR0cHM6Ly93d3cueW91dHViZS5jb20vd2F0Y2g_dj1tTE13b2U1cGxEZw==) that David Guetta, Madonna, and Lil Wayne made back in 2009.
    
9.  [@matry/dom](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/25h2h9u294x3xdi3/aHR0cHM6Ly9naXRodWIuY29tL21hdHJ5L2RvbQ==) is a new, early-stage UI library for “imperative JSX.” Instead of following React’s declarative model that treats JSX as the source of truth for your UI, it essentially becomes a query interface for DOM manipulation.
    
10.  [Virgil](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/qvh8h8urwo3d3qhl/aHR0cHM6Ly9naXRodWIuY29tL3RpdHplci92aXJnaWw=) is a brand new, lightweight systems programming language created by Ben Titzer (the co-creator of WebAssembly). Really thoughtful of him to name this new project after my great uncle who used to scare us all at family reunions by pulling out his glass eye.

2. 埃里克·阿拉姆写了关于他和他的团队如何深沉的潜水[Node.js 事件环节延迟](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/dphehmu068xex4tm/aHR0cHM6Ly90cmlnZ2VyLmRldi9ibG9nL2V2ZW50LWxvb3AtbGFn)是的

3. React Rally是迄今为止最受欢迎的React confs之一,他们正在提供[25%的折扣给所有Bytes订阅者通过这个链接](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/e0hph0u0vqz7zlc8/aHR0cHM6Ly90aS50by9yZWFjdC1yYWxseS9yZWFjdC1yYWxseS0yMDI0L2Rpc2NvdW50L0J5dGVz)在8月12日至14日来到犹他州帕克城,并从Chris Sev、Kent C. Dodds、Shruti Kapoor和其他许多人那里学习。

4. [工具工具](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/owhkhwur9geweluv/aHR0cHM6Ly9naXRodWIuY29tL3Rvc3MvZXMtdG9vbGtpdA==)是一个新的JavaScript工具库,它将自己定义为更快、更现代化的Lodash版本,这是一种更快、更现代化的 underscore版本。

5. [马克思.css](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/z2hgh7u36pzez8ip/aHR0cHM6Ly9naXRodWIuY29tL2xvd2xpZ2h0ZXIvbWF0Y2hh)是一种纯粹的 CSS 库,旨在设计类似默认浏览器风格表的 HTML 元素。*欲*喜欢马查,它总是对我来说像草汁一样的味道。

6. [反应,可视化](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/p8hehquz6ge4e7sq/aHR0cHM6Ly9yZWFjdC5nZy92aXN1YWxpemVkP3M9Ynl0ZXM=)这是我们温和的介绍反响,但可视化。

7. 桑德斯[JavaScript 错误和性能监控工具](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/x0hph3uw29memnf5/aHR0cHM6Ly9zZW50cnkuaW8vZm9yL2phdmFzY3JpcHQv)你得到React 19的支持,数十个图书馆的自动配置,以及一个单行设置,使它非常容易开始(免费)。

8. [流浪者](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/6qhehou7rn4l47bo/aHR0cHM6Ly9mbGl0dGVyLnBhZ2VzLmRldi8=)这是一个JavaScript数据可视化库,它使用类似Flutter的语法 - 即没有人要求的交叉。[克罗斯沃弗洛普](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/kkhmh2ulw6pnpnhl/aHR0cHM6Ly93d3cueW91dHViZS5jb20vd2F0Y2g_dj1tTE13b2U1cGxEZw==)这是大卫·盖塔,麦当娜和莉尔·韦恩在2009年重新创作的。

9. [马德里 / Dom](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/25h2h9u294x3xdi3/aHR0cHM6Ly9naXRodWIuY29tL21hdHJ5L2RvbQ==)它是一个新的,初级阶段的用户界面库,用于“强制性的JSX”。而不是遵循React的声明模型,该模型将JSX视为您的用户界面的真相来源,它基本上成为DOM操作的查询界面。

10. [维吉尔](https://click.convertkit-mail4.com/92udw7d2rkanhqog8q3h9h0vvld33bw/qvh8h8urwo3d3qhl/aHR0cHM6Ly9naXRodWIuY29tL3RpdHplci92aXJnaWw=)这是一个全新的,轻量级的系统编程语言由Ben Titzer(WebAssembly的联合创始人)创建的。他真的很想念这个新项目以我的大叔的名字命名,他曾经通过抽出他的玻璃眼来吓我们所有人在家庭聚会。
