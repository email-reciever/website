## Pop Quiz: Answer

#### Sponsored by [Datadog](https://click.convertkit-mail4.com/68u0wq06r5a8hk490qoi9up34qkkkh9/48hvh7ur4wqv4pbx/aHR0cHM6Ly93d3cuZGF0YWRvZ2hxLmNvbS9yZXNvdXJjZXMvZGVzaWduaW5nLWNsb3VkLWFyY2hpdGVjdHVyZS8_dXRtX2NhbXBhaWduPWRnLWNvcmVwbGF0Zm9ybS13dy1jbG91ZC1hcmNoaXRlY3R1cmUtYnl0ZXMmdXRtX21lZGl1bT1uZXdzbGV0dGVyJnV0bV9zb3VyY2U9Ynl0ZXNkZXY=)

The answer is 65. MDN explains “Non-strict functions that are passed rest, default, or destructured parameters will not sync new values assigned to parameters in the function body with the arguments object.”

If we remove the default parameters we get a different answer:

```js
function calculateTotalCost(mealCost, taxRate, tip) &lcub;
  arguments[1] = 0.15;
  arguments[2] = arguments[2] + 5;

  let totalCost = mealCost + mealCost * taxRate + tip;
  return totalCost;
}

let total = calculateTotalCost(50, undefined, 10);
console.log(total);
```

The answer with no default params is `72.5` (`50` + `7.5` + `15`).

## 小测验：答案

#### \[Datadog]\(https://click.convertkit-mail4.com/68u0wq06r5a8hk490qoi9up34qkkkh9/48hvh7ur4wqv4pbx/aHR0cHM6Ly93d3cuZGF0YWRvZ2hxLmNvbS9yZXNvdXJjZXMvZGVzaWduaW5nL WNsb3VkLWFyY2hpdGVjdHVyZS8\_dXRtX2NhbXBhaWduPWRnLWNvcmVwbGF0Zm9ybS13dy1jbG91ZC1hcmNoaXRlY3R1cmUtYnl0ZXMmdXRtX21lZGl1bT1uZXdzbGV0dGVyJn V0bV9zb3VyY2U9Ynl0ZXNkZXY=)

答案是 65。MDN 解释说“传递剩余参数、默认参数或解构参数的非严格函数不会将分配给函数体中的参数的新值与参数对象同步。”

如果我们删除默认参数，我们会得到不同的答案：

```js
function calculateTotalCost(mealCost, taxRate, tip) &lcub;
  arguments[1] = 0.15;
  arguments[2] = arguments[2] + 5;

  let totalCost = mealCost + mealCost * taxRate + tip;
  return totalCost;
}

let total = calculateTotalCost(50, undefined, 10);
console.log(total);
```

没有默认参数的答案是“72.5”（“50”+“7.5”+“15”）。
