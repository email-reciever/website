## Pop Quiz: Answer

#### Sponsored by [query.gg](https://click.convertkit-mail4.com/xmurv9rn37i6hrmvpq2hgu29erlllsn/25h2h9u2k35n2wf3/aHR0cHM6Ly9xdWVyeS5nZz9zPWJ5dGVz)

How would you remove the duplicate elements from this array?

```js
const list = [
  &lcub; name: 'John' }, 
  &lcub; name: 'Sara' },
  &lcub; name: 'Sara' },
  &lcub; name: 'Lynn' },
  &lcub; name: 'Jake' }
];
```

There are a few ways to do this. Your first intuition might be to do something like this.

```js
const uniqueList = Array.from(new Set(list))
```

It’s the right idea since creating a `Set` will ensure our collection only contains _unique_ values and `Array.from` allows us to create a new, shallow-copied array. Unfortunately, `Sets` only enforce uniqueness for primitive values, but our list is full of objects.

Instead, we can do something like this.

```js
const map = new Map();

list.forEach(item => &lcub;
	map.set(item.name, item)
});

const uniqueList = Array.from(map.values())
```

Notice we’re using a `Map` here. Since `Map`’s preserve insertion order, our `uniqueList` will have the same order as the original list, but without the duplicates since we’re using `name` as the key.

Another fancy approach is using `filter` with `findIndex`,

```js
const uniqueList = list.filter((item, index) => 
  list.findIndex((&lcub;name}) => 
    name === item.name
  ) === index
);
```

Or using `.reduce` and `.map`,

```js
const uniqueList = Object.keys(
  list.reduce((acc, cur) => &lcub;
    acc[cur.name] = cur.name;
    return acc;
  }, &lcub;}),
).map((name) => (&lcub; name }));
```

Or one of the other various ways that I’m sure you’ll all let me know about 🫶.

## 小测验：答案

#### 由 [query.gg](https://click.convertkit-mail4.com/xmurv9rn37i6hrmvpq2hgu29erlllsn/25h2h9u2k35n2wf3/aHR0cHM6Ly9xdWVyeS5nZz9zPWJ5dGVz) 赞助

如何从该数组中删除重复元素？

```js
const list = [
  &lcub; name: 'John' }, 
  &lcub; name: 'Sara' },
  &lcub; name: 'Sara' },
  &lcub; name: 'Lynn' },
  &lcub; name: 'Jake' }
];
```

有几种方法可以做到这一点。您的第一直觉可能是做这样的事情。

```js
const uniqueList = Array.from(new Set(list))
```

这是正确的想法，因为创建“Set”将确保我们的集合仅包含*唯一*值，而“Array.from”允许我们创建一个新的浅复制数组。不幸的是，“集合”仅强制原始值的唯一性，但我们的列表充满了对象。

相反，我们可以做这样的事情。

```js
const map = new Map();

list.forEach(item => &lcub;
	map.set(item.name, item)
});

const uniqueList = Array.from(map.values())
```

请注意，我们在这里使用“Map”。由于“Map”保留插入顺序，因此我们的“uniqueList”将具有与原始列表相同的顺序，但没有重复项，因为我们使用“name”作为键。

另一种奇特的方法是将“filter”与“findIndex”一起使用，

```js
const uniqueList = list.filter((item, index) => 
  list.findIndex((&lcub;name}) => 
    name === item.name
  ) === index
);
```

或者使用`.reduce`和`.map`，

```js
const uniqueList = Object.keys(
  list.reduce((acc, cur) => &lcub;
    acc[cur.name] = cur.name;
    return acc;
  }, &lcub;}),
).map((name) => (&lcub; name }));
```

或者我相信你们都会让我知道的其他各种方式之一。
