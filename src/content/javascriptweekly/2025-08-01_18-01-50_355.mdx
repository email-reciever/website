This v2 release previews a big step forward with a new notebook file format based on HTML and, for the first time, support for true vanilla JavaScript, complete with the ability to import libraries with `import`. [Here’s another neat example](https://javascriptweekly.com/link/172650/d076523b1d) showing off the potential. There are a lot of parts here, so dig in.
