Jarred Sumner

[npm v12 Released: Install Scripts are Off by Default](https://javascriptweekly.com/link/187846/d076523b1d "github.blog") — npm 12 is now GA with [many breaking changes](https://javascriptweekly.com/link/187847/d076523b1d). The big change is lifecycle scripts no longer run by default, but here are [more practical implications to consider](https://javascriptweekly.com/link/187848/d076523b1d). It may also be worth upgrading [via 11.18.0](https://javascriptweekly.com/link/187849/d076523b1d) first.

**IN BRIEF:**

*   TC39's 115th meeting takes place next week. [Here's the agenda](https://javascriptweekly.com/link/187850/d076523b1d), including [Await Dictionary](https://javascriptweekly.com/link/187851/d076523b1d) aiming for Stage 3, plus [Decimal](https://javascriptweekly.com/link/187852/d076523b1d), Import Defer and [Map.take](https://javascriptweekly.com/link/187853/d076523b1d).
    
*   [Next.js now has a formal security release program](https://javascriptweekly.com/link/187854/d076523b1d) with monthly announcements and scheduled releases. The first is expected on July 20.
    
*   ⚠️ The `jscrambler` npm package [suffered a supply chain attack at the weekend](https://javascriptweekly.com/link/187855/d076523b1d) which Socket detected within six minutes.
    
*   The [Porffor](https://javascriptweekly.com/link/187856/d076523b1d) ahead-of-time JavaScript compiler [is now self-hosting](https://javascriptweekly.com/link/187857/d076523b1d) (can compile itself with itself).
    
*   [Vercel has acquired](https://javascriptweekly.com/link/187858/d076523b1d) the team behind [Better Auth](https://javascriptweekly.com/link/187859/d076523b1d), the popular auth library.
    

**RELEASES:**

*   ⭐ [TypeScript 7.0](https://javascriptweekly.com/link/187860/d076523b1d) – The final release of the "10x faster" Go-powered version of the TypeScript compiler is here. A (temporary) lack of a full API means many users are recommended to [stay on 6.0](https://javascriptweekly.com/link/187861/d076523b1d) for now, though.
    
*   [Node.js 26.5.0 (Current)](https://javascriptweekly.com/link/187862/d076523b1d) – Adds support for [importing text files in ES modules](https://javascriptweekly.com/link/187863/d076523b1d) via import attributes, plus `blob.textStream()`.
    
*   [ESLint 10.7](https://javascriptweekly.com/link/187864/d076523b1d), [Storybook 10.5](https://javascriptweekly.com/link/187865/d076523b1d), [Vite DevTools 0.4](https://javascriptweekly.com/link/187866/d076523b1d)
    

📖  Articles and Videos

[`Proxy` and `Reflect`: Redefining How Objects Fundamentally Work](https://javascriptweekly.com/link/187867/d076523b1d "piccalil.li") — A deep dive into intercepting the internal operations of JavaScript objects, covering why `Reflect` exists at all, and ending with a small reactive state system (essentially [how Vue 3's reactivity works](https://javascriptweekly.com/link/187868/d076523b1d)).

Mat Marquis

[In Defense of Polyfills](https://javascriptweekly.com/link/187869/d076523b1d "lea.verou.me") — Prolific standards editor Anne van Kesteren said polyfilling is harmful. Lea disagrees, arguing _"ponyfills are just userland libraries with better marketing"_ and a web without polyfills would be a stagnant one.

Lea Verou

[Sentry + GitHub: Bugs Diagnosed and Fixed, Live](https://javascriptweekly.com/link/187870/d076523b1d "sentry.io") — Sentry catches it, Seer diagnoses it, Copilot writes the fix. Watch Sentry and GitHub break a script live July 15th.

Sentry sponsor

[Collaborative Editing in Wordgard](https://javascriptweekly.com/link/187871/d076523b1d "marijnhaverbeke.nl") — Marijn recently released his new [Wordgard](https://javascriptweekly.com/link/187872/d076523b1d) rich text editor (and ProseMirror successor). Here's his third take on collaborative editing in eleven years, still based on operational transformation (OT) rather than CRDTs.

Marijn Haverbeke

💡 We missed [Marijn's Wordgard release post](https://javascriptweekly.com/link/187873/d076523b1d) in last week's Wordgard feature. He explains why and how he built it (no LLMs!) and why he's _not_ taking pull requests.

[Reverse Engineering ChatGPT's Web Stack](https://javascriptweekly.com/link/187874/d076523b1d "performance.dev") — The author spent days breaking down ChatGPT's page source, bundles, and requests. React Router 7, TanStack Query, Radix, and ProseMirror all make an appearance.
