[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/ugbwvfrjtbd4yzmc8mbj.jpg)](https://javascriptweekly.com/link/174277/d076523b1d)

[SpessaSynth: SoundFont2-Based MIDI Player and Synthesizer](https://javascriptweekly.com/link/174277/d076523b1d "github.com") — If you think typical browser-played MIDI files sound terrible, you’re right – but try this! It uses a SoundFont sample driven approach to play MIDI files in a predictable fashion, and also includes an editor/visualizer if you need it. [The live demo](https://javascriptweekly.com/link/174278/d076523b1d) is quite striking.

Spessasus

While we're on the topic, here are some other neat music related JavaScript projects we've enjoyed over the years:

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/e074yzw1bgpvb4fvef1z.jpg)](https://javascriptweekly.com/link/174279/d076523b1d)

*   [alphaTab](https://javascriptweekly.com/link/174279/d076523b1d) – A full on music notation and guitar tab rendering library for building complete musical apps _(above)_.
    
*   [chiptune3.js](https://javascriptweekly.com/link/174280/d076523b1d) – A bit like SpressaSynth _(also above)_ but for playing _module file_ music. [Live demo.](https://javascriptweekly.com/link/174281/d076523b1d)
    
*   [Tone.js](https://javascriptweekly.com/link/174282/d076523b1d) – An easy way to use the Web Audio API to create music in the browser. Someone used it to [recreate the famous THX 'deep note' sound.](https://javascriptweekly.com/link/174283/d076523b1d)
    
*   🎸 [SVGuitar](https://javascriptweekly.com/link/174284/d076523b1d) – A library to render SVG guitar chord charts.
    
*   [JZZ.js](https://javascriptweekly.com/link/174285/d076523b1d) – A JavaScript MIDI library that hides a lot of complexity behind its chained syntax when you need to work directly with MIDI. (The keyboard logo at the top left of its homepage is a neat easter egg.)
    
*   [Strudel](https://javascriptweekly.com/link/174286/d076523b1d) – A live-coding environment in the browser for generating pieces of music with simple, chained JavaScript expressions.

[SpessaSynth: SoundFont2-Based MIDI Player and Synthesizer](https://javascriptweekly.com/link/174277/d076523b1d "github.com") — If you think typical browser-played MIDI files sound terrible, you’re right – but try this! It uses a SoundFont sample driven approach to play MIDI files in a predictable fashion, and also includes an editor/visualizer if you need it. [The live demo](https://javascriptweekly.com/link/174278/d076523b1d) is quite striking.

While we're on the topic, here are some other neat music related JavaScript projects we've enjoyed over the years:

[![](https://res.cloudinary.com/cpress/image/upload/w_1280,e_sharpen:60,q_auto/e074yzw1bgpvb4fvef1z.jpg)](https://javascriptweekly.com/link/174279/d076523b1d)

* [alphaTab](https://javascriptweekly.com/link/174279/d076523b1d) – A full on music notation and guitar tab rendering library for building complete musical apps *(above)*.

* [chiptune3.js](https://javascriptweekly.com/link/174280/d076523b1d) – A bit like SpressaSynth *(also above)* but for playing *module file* music. [Live demo.](https://javascriptweekly.com/link/174281/d076523b1d)

* [Tone.js](https://javascriptweekly.com/link/174282/d076523b1d) – An easy way to use the Web Audio API to create music in the browser. Someone used it to [recreate the famous THX 'deep note' sound.](https://javascriptweekly.com/link/174283/d076523b1d)

* 🎸 [SVGuitar](https://javascriptweekly.com/link/174284/d076523b1d) – A library to render SVG guitar chord charts.

* [JZZ.js](https://javascriptweekly.com/link/174285/d076523b1d) – A JavaScript MIDI library that hides a lot of complexity behind its chained syntax when you need to work directly with MIDI. (The keyboard logo at the top left of its homepage is a neat easter egg.)

* [Strudel](https://javascriptweekly.com/link/174286/d076523b1d) – A live-coding environment in the browser for generating pieces of music with simple, chained JavaScript expressions.
