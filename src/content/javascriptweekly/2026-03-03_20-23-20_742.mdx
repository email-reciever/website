**IN BRIEF:**

*   [_The React Foundation_ has officially launched](https://javascriptweekly.com/link/181404/d076523b1d) taking over ownership of React, React Native and JSX. It has a board of eight founding members (including Meta) with Seth Webster, former head of React at Meta, as exec director.
    
*   It's not formal yet, but [this preview post reveals how Node.js's release schedule is changing](https://javascriptweekly.com/link/181405/d076523b1d) later this year (one major release per year, every release to be LTS, with no more odd/even distinction).
    
*   📢 Monthly roundups have landed from [Svelte](https://javascriptweekly.com/link/181406/d076523b1d), [ViteLand / VoidZero](https://javascriptweekly.com/link/181407/d076523b1d), and [Astro.](https://javascriptweekly.com/link/181408/d076523b1d)

*   [The Navigation API is now Baseline Newly Available](https://javascriptweekly.com/link/181410/d076523b1d) across all major browsers.
    

**RELEASES:**

* 🎉 Three weeks ago we featured [npmx.dev](https://javascriptweekly.com/link/181464/d076523b1d), a new, fast way to browse and search the official npm registry. Things have got more serious this week with [a full announcement post about the site's alpha release](https://javascriptweekly.com/link/181463/d076523b1d), coupled with a [flurry](https://javascriptweekly.com/link/181465/d076523b1d) of [enthusiastic](https://javascriptweekly.com/link/181466/d076523b1d) [blog](https://javascriptweekly.com/link/181467/d076523b1d) [posts](https://javascriptweekly.com/link/181468/d076523b1d) from around [the community](https://javascriptweekly.com/link/181469/d076523b1d) celebrating the project. I can't remember the last time a JavaScript project attracted so many blog posts at the same time.

* [Locutus](https://javascriptweekly.com/link/181451/d076523b1d) is a curious project that offers TypeScript ports of standard libraries from fifteen other programming languages (e.g. PHP, Go, Python, Ruby).

* [The results of the *State of React Native 2025* survey](https://javascriptweekly.com/link/181452/d076523b1d) have been released by Software Mansion and Devographics.

* [Detailed notes for performing a TypeScript 5.x to 6.0 migration.](https://javascriptweekly.com/link/181453/d076523b1d) The author suggests it might be handy to feed to an AI agent of your choice.

* [An interactive version](https://javascriptweekly.com/link/181470/d076523b1d) of the [famous xkcd comic](https://javascriptweekly.com/link/181471/d076523b1d) about single-person dependencies, courtesy of [p5.js.](https://javascriptweekly.com/link/181472/d076523b1d)

* The [Drizzle ORM](https://javascriptweekly.com/link/181473/d076523b1d) team has [joined PlanetScale.](https://javascriptweekly.com/link/181474/d076523b1d)

* [A list of little, fun programming languages.](https://javascriptweekly.com/link/181454/d076523b1d)
