[Svelte 5 is Alive](https://javascriptweekly.com/link/161451/d076523b1d "svelte.dev") — The long awaited next major release of [Svelte](https://javascriptweekly.com/link/161452/d076523b1d), the compiler-driven JS UI framework, is the _“most significant release in the project’s history”_, while remaining largely backwards compatible. A big addition is _runes_ for explicitly declaring reactive state, but there’s [much more besides.](https://javascriptweekly.com/link/161453/d076523b1d) The official [svelte.dev](https://javascriptweekly.com/link/161452/d076523b1d) site has also [undergone a big rebuild](https://javascriptweekly.com/link/161454/d076523b1d) to act as an ‘omnisite’ for all things Svelte.

The Svelte Team

Key features and virtualized rendering are included in the grid module. Pick and choose special features to keep your app small. Built for JavaScript, extended to Angular, React, and Vue.

[GenAIScript: Microsoft's Generative AI Scripting Environment](https://javascriptweekly.com/link/161399/d076523b1d "microsoft.github.io") — Microsoft is really poking at AI from all angles. [TypeChat](https://javascriptweekly.com/link/161400/d076523b1d) introduced a type-safe way to talk to LLMs; now _GenAIScript_ offers a JavaScript-powered way to programmatically assemble prompts and process responses. They claim it _“brings essential LLM prompt tooling into a cohesive scripting environment.”_

[Next.js 15 Released](https://javascriptweekly.com/link/161458/d076523b1d "nextjs.org") — It’s a big week for the popular (some may even argue _default_) React framework with [Next.js Conf](https://javascriptweekly.com/link/161459/d076523b1d) starting today, as well as this release. It includes a codemod CLI for easier upgrading, async request APIs, alignment with React 19, and more.

**RELEASES:**

*   ⚛️ [React Native 0.76](https://javascriptweekly.com/link/161406/d076523b1d) – A _big_ release. The so-called ['New Architecture' is now used by default](https://javascriptweekly.com/link/161407/d076523b1d) (so you can now use all the modern React features in RN too), the React Native DevTools are stable, and build times are faster than ever.
    
*   🥖 [Bun](https://javascriptweekly.com/link/161408/d076523b1d) is on a _roll.._ with three releases in the past week. [v1.1.31](https://javascriptweekly.com/link/161409/d076523b1d) added support for `node:http2` server and gRPC, [v1.1.32](https://javascriptweekly.com/link/161410/d076523b1d) added Node's `crypto.hash` method, and [v1.1.33](https://javascriptweekly.com/link/161411/d076523b1d) was a bug-fix release.
    
*   [Express.js 5.0](https://javascriptweekly.com/link/161412/d076523b1d) – Landed a few weeks ago, but now there's an official post explaining the state of the project which has now [passed a comprehensive third party security audit.](https://javascriptweekly.com/link/161413/d076523b1d)
    
*   [Medusa 2.0](https://javascriptweekly.com/link/161414/d076523b1d) – A Node.js-powered ecommerce platform.
    
*   [React Compiler Beta](https://javascriptweekly.com/link/161460/d076523b1d), [Turborepo 2.2](https://javascriptweekly.com/link/161461/d076523b1d), [ESLint v9.13.0](https://javascriptweekly.com/link/161415/d076523b1d), [Deno 2.0.2](https://javascriptweekly.com/link/161416/d076523b1d)
