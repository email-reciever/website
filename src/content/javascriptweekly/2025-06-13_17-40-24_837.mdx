Boshen Chen and Cameron Clark

[pnpm 10.12 Introduces an Experimental Global Virtual Store](https://javascriptweekly.com/link/170461/d076523b1d "socket.dev") — [pnpm](https://javascriptweekly.com/link/170462/d076523b1d) has long been prized for its speed and efficiency over `npm`. [v10.12](https://javascriptweekly.com/link/170463/d076523b1d) takes things further introducing a ‘global virtual store’ that `node_modules` symlinks to, enabling projects to share dependencies without them being installed numerous times.

Sarah Gooding (Socket)

**IN BRIEF:**

*   **Surfin' Safari?** Apple's WWDC25 event this week saw numerous developments that _aren't_ 'liquid glass' related, including [a look at what's new in Safari 26 (beta)](https://javascriptweekly.com/link/170435/d076523b1d) including support for pattern modifiers in `RegExp` objects.
    
*   A writeup of [a vulnerability found in OpenPGP.js.](https://javascriptweekly.com/link/170478/d076523b1d)
    
*   Matteo Collina gives us [a public service announcement around the end-of-life of some recent Node.js versions](https://javascriptweekly.com/link/170432/d076523b1d) and some tips on upgrading to v22.
    
*   [Gleam](https://javascriptweekly.com/link/170433/d076523b1d) is an easy to read/write language targeting both Erlang and JS runtimes. The big news is that [Gleam compiled to JS is now 30% faster.](https://javascriptweekly.com/link/170434/d076523b1d)
    

**RELEASES:**

*   [H3 v2 Beta](https://javascriptweekly.com/link/170464/d076523b1d) – Cross-runtime Web standards focused HTTP server framework.
    
*   [Node.js v24.2 (Current)](https://javascriptweekly.com/link/170436/d076523b1d) – `import.meta.main` is a new boolean value available in ES modules that tells you if the current module was the entry point of the current process.
    
*   [Visual Studio Code May 2025](https://javascriptweekly.com/link/170437/d076523b1d) – Big enhancements to MCP support, but VS Code extensions [can now](https://javascriptweekly.com/link/170438/d076523b1d) also use ES modules.
    
*   [Deno 2.3.6](https://javascriptweekly.com/link/170465/d076523b1d), [Rollup v4.43](https://javascriptweekly.com/link/170439/d076523b1d), [Jasmine 5.8](https://javascriptweekly.com/link/170440/d076523b1d), [Vue DevTools v7 for Firefox](https://javascriptweekly.com/link/170466/d076523b1d)
