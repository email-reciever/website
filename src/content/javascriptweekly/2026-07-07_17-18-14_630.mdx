[Vite+ Beta: A Web Dev Toolchain Behind One Command](https://javascriptweekly.com/link/187516/d076523b1d "voidzero.dev") — Vite+ is the [Vite](https://javascriptweekly.com/link/187517/d076523b1d) team’s ‘unified toolchain’ that brings Vite, Vitest, Oxlint, and similar tools together under a single `vp` command, whether for running a dev server, tests, formatting, or bundling.

VoidZero

💡 Vite+ was originally intended to be a commercial project to fund work on Vite and related projects, but was open sourced under the MIT license earlier this year.

[![](https://res.cloudinary.com/cpress/image/upload/c_limit,w_480,h_480,q_auto/copm/eb4e5347.png)](https://javascriptweekly.com/link/187515/d076523b1d)

[Still Writing Tests Manually? Meticulous AI Is Here](https://javascriptweekly.com/link/187515/d076523b1d "www.meticulous.ai") — Notion, Dropbox, Wiz and LaunchDarkly now use a testing paradigm they can’t work without. Built by former Palantir engineers, Meticulous automatically creates an evolving suite of E2E UI tests, delivering exhaustive coverage with no developer effort.

Meticulous sponsor

[What's New in ECMAScript 2026](https://javascriptweekly.com/link/187518/d076523b1d "pawelgrzybek.com") — The 876 page [ECMAScript 2026 spec](https://javascriptweekly.com/link/187519/d076523b1d) was approved last week, and this round-up covers several main new features that made the cut, from `Array.fromAsync` to native `Uint8Array` Base64/Hex conversion, with examples for each. All are already in browsers and runtimes (except `Math.sumPrecise` in Node) so you can likely use them today.

Pawel Grzybek

**IN BRIEF:**

*   🕹️ [js1024, the annual 1KB code-golf competition, is open for entries](https://javascriptweekly.com/link/187520/d076523b1d). The theme is 'dreaming', and you've got 1,024 bytes to make something neat. Here's [last year's winner](https://javascriptweekly.com/link/187521/d076523b1d) to get a feel for the caliber.
    
*   There's [a TC39 proposal to standardize a `code` property on `Error`](https://javascriptweekly.com/link/187522/d076523b1d) (long a de-facto Node convention). Its champion, James Snell, hopes to advance it at this month's plenary.
    

**RELEASES:**

*   [vinext 1.0 Beta](https://javascriptweekly.com/link/187523/d076523b1d) – Cloudflare's Vite plugin reimplementing Next.js's API surface.
    
*   [webpack-dev-server 6.0](https://javascriptweekly.com/link/187524/d076523b1d) – The official webpack dev server with live reloading upgrades to Express 5 and native ESM.
    
*   [Dependency Cruiser 18.0](https://javascriptweekly.com/link/187525/d076523b1d) – Tool to validate and graph project dependencies.
    
*   [npm 11.18.0](https://javascriptweekly.com/link/187526/d076523b1d), [pnpm 11.10](https://javascriptweekly.com/link/187527/d076523b1d)

[Working with AI: A Concrete Example](https://javascriptweekly.com/link/187528/d076523b1d "htmx.org") — The creator of [htmx](https://javascriptweekly.com/link/187529/d076523b1d) reflects on the role of AI in his work with a practical example of using Claude to investigate a bug report. He also touches on how AI fits in with being an ‘older developer’, a topic I rarely see brought up.

Carson Gross

Rick Zhang (Medal)

[Streaming Real-Time HDR Reflections to Every Device with Miris](https://javascriptweekly.com/link/187531/d076523b1d "www.miris.com") — Miris built a WebXR car configurator that streams a 1.2GB asset in real-time with HDR reflections across any device.

Miris sponsor

📢  Elsewhere in the ecosystem

*   🤖 [Safari has introduced an MCP server](https://javascriptweekly.com/link/187553/d076523b1d) in its latest preview release, providing agents with a way to connect to a Safari window and emulate the user experience while debugging.
    
*   [`shadcn/ui` has switched to Base UI as its default component library](https://javascriptweekly.com/link/187555/d076523b1d) (from Radix). Radix isn't going away, but [Base UI](https://javascriptweekly.com/link/187556/d076523b1d) is recommended for new projects.
    
*   [PocketJS](https://javascriptweekly.com/link/187554/d076523b1d) is a project that brings a JavaScript VM, Vue Vapor, and Solid UI into an 8MB memory budget, and can, curiously, run on the Sony PSP, as well as in WASM and natively on macOS.
    
*   📈 [Commit History](https://javascriptweekly.com/link/187557/d076523b1d) is an online tool that creates charts of a user's GitHub commits over time. A nice addition for your [profile README](https://javascriptweekly.com/link/187558/d076523b1d), or check out [Sindre Sorhus's chart](https://javascriptweekly.com/link/187559/d076523b1d) to see the legendary maintainer's consistency over the years.
    
*   🎉 [PGlite](https://javascriptweekly.com/link/187560/d076523b1d), a WebAssembly build of Postgres for use in JavaScript environments, [celebrates reaching 10 million weekly downloads](https://javascriptweekly.com/link/187561/d076523b1d) while explaining the use cases and how to get started with it yourself.
